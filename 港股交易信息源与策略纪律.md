## 八、交易策略框架

### 📈 策略类型选择

| 策略 | 持仓周期 | 信息依赖 | 风险 |
|------|---------|---------|------|
| **超短线（T+0）** | 分钟-小时 | 实时行情、资金流 | 高 |
| **短线波段** | 1-5天 | 技术指标、消息面 | 中高 |
| **趋势跟踪** | 周-月 | 均线、动量 | 中 |
| **价值投资** | 月-年 | 基本面、研报 | 中低 |
| **事件驱动** | 不定 | 公告、新闻 | 高 |

### 🎯 港股特有策略

#### 1. 南向资金跟踪策略
```
买入条件:
- 南向资金连续3天净流入 > 50亿
- 个股被南向大幅增持
- 技术面不破位

卖出条件:
- 南向资金转为净流出
- 持仓被大幅减少
```

#### 2. 主力成本策略（VWAP）
```
买入条件:
- 主力净流入 > 500万
- 现价 ≤ 20日VWAP（低于主力成本）
- 散户卖+主力买

卖出条件:
- 现价 > 20日VWAP × 115%
- 主力连续3天净流出
```

#### 3. 沽空信号策略
```
买入信号（做多）:
- 沽空比例连续下降
- 沽空后股价企稳（空头回补）

卖出/观望信号:
- 沽空比例骤升至20%+
- 大户持续增加沽空持仓
```

## 九、交易纪律（最重要！）

### 🚨 仓位管理铁律

| 规则 | 执行标准 |
|------|---------|
| **单只股票上限** | ≤ 总资金的 20% |
| **单个行业上限** | ≤ 总资金的 30% |
| **总仓位控制** | 牛市80%，震荡60%，熊市30% |
| **新股/次新股** | ≤ 总资金的 10% |
| **高风险标的** | ≤ 总资金的 5% |

### 🛑 止损纪律

| 情况 | 止损线 | 执行 |
|------|-------|------|
| **日内交易** | -2% | 即时止损 |
| **短线波段** | -5% | 当日收盘前止损 |
| **趋势交易** | -8% | 收盘跌破止损 |
| **价值投资** | -15% 或基本面恶化 | 分批止损 |
| **追高买入** | -3% | 严格止损 |

### 💰 止盈纪律

| 盈利幅度 | 操作 |
|---------|------|
| +10% | 减仓1/3，上移止损到成本 |
| +20% | 减仓1/3，止损上移到+10% |
| +30% | 再减1/3，止损上移到+20% |
| +50% | 清仓或保留底仓 |

### ⏰ 时间纪律

| 规则 | 说明 |
|------|------|
| **不追开盘** | 开盘30分钟内只观察不操作 |
| **不追尾盘** | 收盘30分钟内谨慎操作 |
| **周末不持仓** | 高风险标的周五减仓 |
| **长假清仓** | 国庆/春节前清仓观望 |
| **财报前减仓** | 业绩公布前减持或对冲 |

### 🧠 心理纪律

| 情况 | 应对 |
|------|------|
| 连续亏损3次 | 停止交易1天，复盘 |
| 单日亏损>5% | 当日不再开新仓 |
| 情绪激动时 | 离开屏幕，冷静后再决策 |
| 想"回本"时 | 最危险的心态，必须止损 |
| 过度自信时 | 减小仓位，保持谨慎 |

## 十、每日操作流程

### 🌅 开盘前（8:00-9:30）

```
□ 查看隔夜美股表现
□ 查看A股期货（IF/IC）
□ 阅读重要财经新闻
□ 查看南向资金预期
□ 检查持仓股公告
□ 制定当日交易计划
```

### 📈 盘中（9:30-16:00）

```
09:30-10:00  观察开盘情况，不急于操作
10:00-11:30  确认主力动向，执行计划
12:00-13:00  午间复盘，调整策略
13:00-15:30  跟踪资金流向，灵活应对
15:30-16:00  收盘前评估，决定是否隔夜
```

### 🌙 收盘后（16:00-20:00）

```
□ 记录当日交易明细
□ 计算盈亏及总结
□ 分析做对/做错的原因
□ 查看收盘后公告
□ 更新持仓评估表
□ 制定次日计划
```
