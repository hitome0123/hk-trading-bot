## 一、策略定义

**一句话**：买入热门板块中跌惨了的前期强势股（非恒科成分股），等利好消息爆拉。

### 策略标签

| 标签 | 说明 |
|------|------|
| 类型 | 超跌反弹 + 事件驱动 |
| 周期 | 波段（1-10天） |
| 频率 | 低频（每周1-3次） |
| 风险 | 中高 |

## 八、策略优化

### 提高胜率的方法

| 方法 | 说明 |
|------|------|
| **等消息面** | 不要纯技术面抄底，要有催化剂预期 |
| **看板块联动** | 板块内龙头先涨，跟随股补涨 |
| **分批建仓** | 不要一把梭，分2-3次买入 |
| **设时间止损** | 超过7天不反弹就走 |

### 常见错误

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| 接飞刀 | 越买越跌 | 等企稳信号再买 |
| 抄底成分股 | 弹性不够 | 只选小市值非成分股 |
| 重仓单只 | 一把亏光 | 单只最多30% |
| 不止损 | 深套 | 跌8%必须走 |
| 利好兑现不卖 | 回吐利润 | 利好出来先卖一半 |
