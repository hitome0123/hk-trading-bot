# 盘前资讯监控系统部署说明

## 部署时间
2026-02-22

## 系统概述
每日08:30自动扫描自选股隔夜变化，识别利好/利空信号，推送重要消息到Telegram。

## 核心功能

### 1. 隔夜变化监控
- **监控对象**: 154只自选股（扫描前50只）
- **检测指标**: 隔夜涨跌幅
- **重要信号**: 涨跌幅≥3%

### 2. 板块关联分析
自动识别个股所属板块:
- 人形机器人
- AI大模型
- GPU芯片
- 互联网
- 港股科技
- 新能源车
- 军工
- 生物医药
- 芯片设备
- 云计算

### 3. 利好/利空识别（关键词分析）

**利好关键词:**
- 业绩: 业绩增长、盈利、营收增长、超预期
- 合作: 签约、中标、合作、订单、大单
- 技术: 研发成功、新产品、技术突破、专利
- 资本: 增持、回购、分红、派息
- 评级: 上调评级、买入评级、推荐

**利空关键词:**
- 业绩: 业绩下滑、亏损、业绩预警
- 违规: 违规、处罚、调查、立案
- 减持: 减持、质押、套现
- 评级: 下调评级、卖出评级
- 其他: 利空、大跌、跌停、诉讼

### 4. Telegram推送报告
自动生成盘前报告，包含:
- 🟢 隔夜上涨个股 (Top 5)
- 🔴 隔夜下跌个股 (Top 5)
- 📈 板块分布统计

## 运行时间表

### 每日定时执行
- **执行时间**: 每天 08:30
- **执行条件**: 仅交易日（周一~周五）
- **周末处理**: 自动跳过，等待下周一

### 执行流程
```
08:30 → 检查是否交易日
       ├─ 是 → 连接富途 → 加载自选股 → 扫描50只股票 → 生成报告 → 推送Telegram
       └─ 否 → 跳过扫描，等待下一天
```

## 技术细节

### 依赖
- Python 3.x
- futu-api (富途OpenD)
- requests (Telegram推送)

### 环境变量
```
PROTOCOL_BUFFERS_PYTHON_IMPLEMENTATION=python
```

### 重要说明
1. 需要富途OpenD运行在 127.0.0.1:11111
2. 只扫描前50只自选股（避免耗时太长）
3. 仅读取行情数据，不涉及交易操作
4. 周末自动跳过，不执行扫描

## 与其他系统的协同

### 完整监控链路
```
08:30 盘前扫描 (premarket_news_scanner)
  ↓ 发现隔夜异动
  ↓ 推送Telegram提醒
  ↓
09:00 开盘
  ↓
09:00-16:00 板块炒作雷达 (sector_radar)
  ↓ 每10分钟扫描
  ↓ 发现强炒作信号
  ↓ 推送Telegram提醒
```

### 三层监控体系
1. **盘前层**: 隔夜变化预警（08:30）
2. **盘中层**: 板块炒作监控（09:00-16:00，每10分钟）
3. **全天层**: 自选股价格异动（每30分钟）
