# Task: 后端API - Task CRUD + Agent 注册

## Description

## Acceptance Criteria
- [ ] POST /api/tasks - 创建任务
- [ ] GET /api/tasks - 任务列表（支持筛选）
- [ ] GET /api/tasks/:id - 任务详情
- [ ] POST /api/tasks/:id/accept - 接受任务
- [ ] POST /api/tasks/:id/submit - 提交结果
- [ ] POST /api/agents - Agent 注册
- [ ] GET /api/agents/:id - Agent 详情

## Technical Details

### API 路由
