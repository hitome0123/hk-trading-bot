## Technical Details

### 买方 Agent (Trading Agent)
```typescript
// scripts/buyer-agent.ts
async function runBuyerAgent() {
  // 1. 注册 Agent
  await registerAgent({
    name: "HK Trading Bot",
    capabilities: ["trading", "technical_analysis"],
  });

  // 2. 发布任务
  const task = await createTask({
    title: "分析 HK.02577 英诺赛科近期利好",
    description: "搜索最近一周关于英诺赛科的新闻...",
    price: 1.0, // USDC
    deadline: Date.now() + 600000, // 10分钟
  });

  // 3. 等待完成
  await waitForCompletion(task.id);
  console.log("收到研究报告:", task.result);
}
```

### 卖方 Agent (Research Agent)
```typescript
// scripts/seller-agent.ts
async function runSellerAgent() {
  // 1. 注册 Agent
  await registerAgent({
    name: "Perplexity Research Agent",
    capabilities: ["research", "news_analysis"],
  });

  // 2. 监听新任务
  const task = await findMatchingTask(["research", "news"]);

  // 3. 接单
  await acceptTask(task.id);

  // 4. 执行任务（模拟 Perplexity 搜索）
  const result = await executeResearch(task.description);

  // 5. 提交结果
  await submitResult(task.id, result);
}
```

### Demo 场景脚本
```typescript
// scripts/run-demo.ts
async function runFullDemo() {
  // 并行启动两个 Agent
  await Promise.all([
    runBuyerAgent(),
    runSellerAgent(),
  ]);
  console.log("Demo 完成！交易成功");
}
```
