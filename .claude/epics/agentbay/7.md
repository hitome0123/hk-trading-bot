# Task: LLM验证 - GPT-4 验证服务 + 自动释放

## Description
实现 LLM 自动验证服务，当卖方提交结果后，使用 GPT-4 判断是否满足任务要求，验证通过则自动调用合约释放资金。

## Acceptance Criteria
- [ ] GPT-4 验证 Prompt 设计
- [ ] 验证服务 API 端点
- [ ] 验证结果存储（passed/failed + reason）
- [ ] 验证通过后自动调用合约 release()
- [ ] 验证失败后通知卖方

## Technical Details

### 验证 Prompt
```typescript
const verifyPrompt = `
你是一个任务验证系统。请判断提交的结果是否满足任务要求。

## 任务要求
${task.description}

## 验证标准
${task.verification_rules}

## 提交内容
${submission.result}

## 请回答（JSON格式）
{"passed": true/false, "reason": "...", "score": 1-10}
`;
```

### 服务流程
```
1. 卖方提交结果 → submissions 表
2. 触发验证服务
3. 调用 GPT-4 API
4. 解析验证结果
5. 如果 passed:
   - 调用合约 release(taskId)
   - 更新 task status = completed
6. 如果 failed:
   - 更新 submission status = failed
   - 通知卖方原因
```

### 合约调用
```typescript
// 使用平台私钥签名
const platformWallet = new Wallet(PLATFORM_PRIVATE_KEY);
await escrowContract.connect(platformWallet).release(taskId);
```
