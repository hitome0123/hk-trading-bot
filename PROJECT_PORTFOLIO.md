# 港股智能交易系统 (HK Trading Bot)

> 个人独立开发的量化交易分析平台，集成实时行情、技术分析、AI信号生成、自动化推送于一体

---

## 项目概览

| 指标 | 数据 |
|------|------|
| **代码规模** | 122 个 Python 模块，7.2MB |
| **开发周期** | 2 个月（业余时间） |
| **覆盖市场** | 港股 + 加密货币 |
| **监控标的** | 27 只港股（8 个热门板块）+ 10 种加密货币 |
| **数据更新** | 实时（5分钟级别） |

---

## 核心技术栈

```
后端：Python 3.12 + FastAPI + Flask
数据：Futu OpenD API + YFinance + CoinGecko + Binance
分析：NumPy + Pandas（技术指标计算）
自动化：n8n 工作流引擎
推送：钉钉 Webhook + Telegram Bot
部署：本地 + Docker
```

## 核心功能模块

### 1. 技术指标引擎 (`TechnicalIndicators`)
- **实现指标**：EMA(20/50)、RSI(14)、MACD、ATR、KDJ、布林带
- **技术亮点**：纯 NumPy 实现，单次计算 < 10ms
- **应用场景**：趋势判断、超买超卖识别、波动率评估

### 2. 智能入场策略 (`EntryStrategy`)
- **策略类型**：
  - 趋势跟踪：EMA 金叉 + RSI 确认
  - 均值回归：RSI 超卖 + 布林带下轨
  - 突破交易：ATR 扩张 + 成交量放大
- **输出内容**：入场价、止损价、止盈目标、仓位建议

### 3. 风险管理系统 (`RiskManager`)
- **仓位控制**：单笔上限 10,000 HKD，每日最多 5 笔
- **风险约束**：最大回撤 2%、价格区间 1-1000 HKD
- **合规检查**：交易时间验证、黑名单过滤

### 4. 模拟交易系统 (`PaperTrader`)
- **功能**：10万虚拟资金、持仓跟踪、盈亏计算
- **数据持久化**：JSON 存储交易记录

### 5. 四层分析引擎 (`AnalysisEngine`)
```
Layer 1: 形态识别 → K线形态、图表模式
Layer 2: 资金流向 → 机构买卖、北向资金
Layer 3: 相对强度 → 板块对比、大盘联动
Layer 4: AI叙事   → 新闻情绪、社媒热度
```

### 6. 社交媒体热度监控
- **数据源**：东方财富（港股+A股+概念）、淘股吧
- **分析维度**：K值加速度检测（0.3-1.0 为理想入场区）
- **输出频率**：每 10 分钟刷新

### 7. 加密货币模块
- **支持币种**：BTC、ETH、SOL、DOT 等 10 种
- **特色功能**：恐惧贪婪指数、资金费率、24/7 监控
- **策略适配**：针对高波动特性调整止损止盈参数
