## 五、实战操作流程

### 场景1：发现热门股票
```bash
# 1. 先查情绪
python sentiment_hub.py 09880

# 2. 看结果判断
# ✅ positive + 热度>70 → 继续研究
# ❌ negative 或 热度<30 → 放弃

# 3. 深度研究
python gemini_deep_research.py 09880

# 4. 结合研究报告做决策
```

### 场景2：持仓股异动
```bash
# 1. 快速查情绪变化
python sentiment_hub.py 09880

# 2. 看韩国社区（亚洲资金风向）
python -c "
from sentiment_hub import SentimentHub
hub = SentimentHub()
result = hub._dcinside_analyze('优必选', '09880')
print(result.summary if result else '无数据')
"

# 3. 根据情绪决定加仓/减仓/持有
```

### 场景3：寻找补涨机会
```bash
# 1. 看龙头表现
python sentiment_hub.py 02513   # 智谱（AI龙头）

# 2. 如果龙头情绪火热，查同板块
python my_strategy_helper.py laggards

# 3. 研究掉队股
python gemini_deep_research.py 00100  # MiniMax
```

## 八、风险提示

1. **情绪分析仅供参考**，不构成投资建议
2. **设置止损**：单笔最大亏损 2%
3. **分散持仓**：单只股票不超过总仓位 20%
4. **韩国社区滞后**：韩国散户信息可能比机构慢
5. **API 限制**：OpenAI 有调用频率限制，不要刷太频繁
