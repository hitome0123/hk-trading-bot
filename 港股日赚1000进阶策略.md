### 高波动股每日筛选流程

```python
def scan_high_volatility(min_volatility=5):
    """
    筛选日波动率>5%的股票
    """
    candidates = []

    for code in watchlist:
        kline = get_kline(code, 20)

        # 计算20日平均波动率
        avg_volatility = ((kline['high'] - kline['low']) / kline['close']).mean() * 100

        # 计算今日波动率
        today = kline.iloc[-1]
        today_volatility = (today['high'] - today['low']) / today['close'] * 100

        # 量比 (今日成交量/20日均量)
        volume_ratio = today['volume'] / kline['volume'].tail(20).mean()

        if avg_volatility >= min_volatility and volume_ratio >= 1.5:
            candidates.append({
                'code': code,
                'avg_volatility': avg_volatility,
                'today_volatility': today_volatility,
                'volume_ratio': volume_ratio
            })

    return sorted(candidates, key=lambda x: x['avg_volatility'], reverse=True)
```

### 入选条件

必须同时满足:
- [x] 20日平均波动率 ≥ 5%
- [x] 今日量比 ≥ 1.5 (放量)
- [x] 有明确消息/板块催化剂
- [x] 非停牌/退市风险股

## 三、Oliver Velez日内交易法 (新增)

### 2-Minute/5-Minute Momentum Strategy

Oliver Velez专注于日内动量交易，核心原则：

| 原则 | 说明 |
|------|------|
| **趋势确认** | 只在明确趋势方向交易 |
| **动量入场** | 等待强势K线确认后入场 |
| **快进快出** | 目标小利多次，不恋战 |
| **严格止损** | 每笔亏损≤入场价1% |

### 关键K线形态

**做多信号 (买入):**
1. **长阳突破**: 实体>前3根K线平均实体×1.5
2. **量价配合**: 成交量>前5根平均×2
3. **位置确认**: 在支撑位上方

**做空信号 (卖出):**
1. **长阴跌破**: 实体>前3根平均×1.5
2. **放量下跌**: 成交量>前5根平均×2
3. **位置确认**: 跌破支撑位

### Velez入场检查清单

```
□ 大盘方向确认（恒指趋势）
□ 板块强势确认（AI/机器人板块资金流入）
□ 个股突破确认（Williams波动率突破）
□ K线形态确认（长阳/放量）
□ 止损位设定（入场价下方1%）
```

### 入场评分计算

```python
def entry_score(code):
    """
    入场评分系统 (满分100分)
    ≥70分: 立即入场
    50-69分: 观望等待
    <50分: 不操作
    """
    score = 0

    # 1. Minervini (20分)
    if minervini_stage2(code):
        score += 20

    # 2. Livermore (15分)
    if near_support_level(code):
        score += 15
    elif breakout_resistance(code):
        score += 10

    # 3. Williams (20分)
    if williams_breakout_signal(code) == 'BUY':
        score += 20

    # 4. Sperandeo (15分)
    if no_reversal_signal(code):
        score += 15
    elif buy_reversal_signal(code):
        score += 20  # 加分

    # 5. VWAP (15分)
    vs_vwap = current_vs_vwap(code)
    if vs_vwap <= -5:
        score += 15
    elif vs_vwap <= 0:
        score += 10
    elif vs_vwap > 20:
        score -= 10  # 扣分

    # 6. Velez动量 (15分)
    if strong_momentum_candle(code):
        score += 15

    return score
```

## 九、风险警告

### 日赚1000的现实挑战

1. **528%年化收益是极端目标**
   - 全球顶级对冲基金年化约20-30%
   - 个人交易者持续盈利已很难得

2. **高收益必然伴随高风险**
   - 每日2%收益意味着每日可能-2%亏损
   - 连续5天亏损 = -10% = 5000元

3. **交易成本侵蚀利润**
   - 港股印花税: 0.1%
   - 佣金: 约0.05%
   - 每笔来回成本约0.3%

4. **心态是最大敌人**
   - 连续亏损后容易失控
   - 连续盈利后容易膨胀

### 建议的进阶路径

```
第1阶段 (1个月): 目标日赚300元，熟悉系统
第2阶段 (2个月): 目标日赚500元，提高胜率
第3阶段 (3个月): 目标日赚800元，优化仓位
第4阶段 (持续): 目标日赚1000元，稳定盈利
```
