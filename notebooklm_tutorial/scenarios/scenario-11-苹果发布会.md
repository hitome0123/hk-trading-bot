## 核心知识点

### 1. "Buy the Rumor, Sell the News" 原理

**市场心理学**:
```
传闻阶段 (T-14天):
- 市场预期乐观 → 买盘涌入 → 股价上涨 +3-5%

发布会前 (T-3天):
- 预期充分反映 → 获利盘准备离场

发布会当天 (T):
- 利好落地 → "News is out" → 获利了结 → 股价下跌 -1-3%

发布会后 (T+30天):
- 实际产品反馈 → 销售数据 → 真实基本面驱动
```

**关键时间节点**:
- T-14: 传闻开始发酵 (最佳买入期)
- T-7: 预期高峰 (谨慎期)
- T-1: 最后撤退机会 (卖出窗口)
- T: 发布会当天 (高风险, 不参与)

---

### 2. 催化剂类型与影响力

**催化剂分级**:

| 类型 | 影响力 | 持续时间 | 股价波动 | 案例 |
|------|--------|---------|---------|------|
| **产品发布** | ⭐⭐⭐⭐⭐ | 2-4 周 | ±5-10% | iPhone 新品 ✅ |
| **财报发布** | ⭐⭐⭐⭐⭐ | 1-2 周 | ±8-15% | Q4 财报 |
| **并购消息** | ⭐⭐⭐⭐ | 1-3 月 | ±10-30% | 收购 AI 公司 |
| **监管新闻** | ⭐⭐⭐ | 1-2 周 | ±3-8% | App Store 反垄断 |
| **CEO 变动** | ⭐⭐⭐ | 1-2 月 | ±5-15% | 库克退休传闻 |

**苹果发布会特殊性**:
- 全球关注度最高的科技发布会
- 媒体造势周期长 (2-3 周)
- 预期管理艺术 (库克风格: 低调 → 超预期)

### 4. 风险控制

**失败案例分析 - 2018 年**:
- 传闻: iPhone XS 创新不足
- 发布会前: 股价仅涨 +0.5% (vs 平均 +3.5%)
- 发布会后: 股价跌 -2.5%
- **教训**: 如果传闻偏负面, 不要强行布局

**止损策略**:
```
买入价: $175
止损价: $172 (-1.7%)
目标价: $182 (+4%)

风险收益比: 4% / 1.7% = 2.35 (优秀)
```

## 实战决策

### ✅ 买入信号
1. **传闻积极** - 产品创新点清晰 (AI 功能, 新设计)
2. **历史规律支持** - 过去 5 年平均涨 +3.7%
3. **时机正确** - 发布会前 10-14 天 (8/30-9/3)
4. **技术面配合** - 股价突破 20 日均线 + 成交量放大

### ⚠️ 风险提示
1. **产品令人失望** - 如果无重大创新, 股价可能 -5%+
2. **宏观环境恶化** - 美联储加息, 科技股整体下跌
3. **竞争加剧** - 华为 Mate 60 Pro 回归, 中国市场份额下降
4. **估值过高** - PE 30x, 高于历史中位数 25x

### 🎯 操作策略

**策略 A: 标准催化剂交易 (推荐)**
```
买入时机: 8/30-9/3 (发布会前 10-14 天)
买入价: $175-178
持仓期: 10-12 天
卖出时机: 9/11 (发布会前 1 天) 收盘前
目标收益: +3-5%
止损: -2%
```

**策略 B: 激进发布会当天赌博 (不推荐)**
```
买入时机: 9/12 发布会结束后 (如果产品超预期)
风险: 极高 (可能 -5%+)
仅适合: 专业交易员 + 小仓位
```

**策略 C: 长期持有忽略催化剂 (保守)**
```
如果是长期投资者, 无需在意短期催化剂
发布会前后波动 ±5% 对长期收益影响小
继续持有, 等待 Q4 财报
```
