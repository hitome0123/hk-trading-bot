## 核心知识点

### 1. 三表模型识别财务造假

**利润表说谎, 现金流不会**:
```
利润表 (Income Statement):
- 净利润: 100 亿 ✅ (看似盈利)

现金流量表 (Cash Flow Statement):
- 经营现金流: -50 亿 ❌ (实际亏损)

矛盾: 利润 +100 亿, 现金流 -50 亿
→ 差异 150 亿 = 应收账款 + 存货 ⚠️

结论: "账面盈利" 都是赊销 + 存货堆积, 现金没进来!
```

**三表联动分析**:
```python
# Step 1: 利润表 → 净利润 100 亿
# Step 2: 现金流表 → 经营现金流 -50 亿
# Step 3: 资产负债表 → 应收账款 +150 亿, 存货 +80 亿

推理:
100 亿净利润 - 150 亿应收增加 - 80 亿存货增加 = -130 亿
+ 其他调整 80 亿 = -50 亿经营现金流 ✅

结论: 利润全是账面, 现金一分没收到!
```

---

### 2. 现金流危机的四大信号

**信号 1: 现金流转化率 < 70%**
```python
转化率 = 经营现金流 / 净利润

隆基绿能:
转化率 = -50 / 100 = -50% ❌

评级标准:
- > 100%: 优秀 (现金流比利润还多)
- 70-100%: 良好
- 50-70%: 中等 ⚠️
- < 50%: 差 (账面盈利, 现金亏损)
- < 0%: 危机 (流血) ❌
```

**信号 2: 应收账款增速 > 营收增速**
```python
营收增速: +5%
应收账款增速: +80%
比值: 80 / 5 = 16x ❌

健康标准: 应收增速 ≤ 营收增速 (比值 < 1.0)
隆基: 16x ❌ 严重异常, 可能大量坏账
```

**信号 3: 存货周转天数暴增**
```python
存货周转天数 = (存货 / 营业成本) × 365

隆基绿能:
存货周转天数 = (200 / 1,000) × 365 = 73 天
vs 去年: 45 天
增加: +28 天 (+62%) ⚠️

行业标准: 45 天
隆基: 73 天 ❌ 存货严重积压
```

**信号 4: 资产负债率 > 60%**
```python
资产负债率 = 负债 / 资产

隆基绿能:
资产负债率 = 500 / 800 = 62.5% ⚠️

评级标准:
- < 40%: 安全
- 40-50%: 健康
- 50-60%: 中等 ⚠️
- 60-70%: 偏高 ⚠️
- > 70%: 危险 ❌

隆基: 62.5% ⚠️ 接近危险线
```

### 3. 光伏行业特殊风险

**产能过剩 → 价格战 → 利润虚假**:
```
2023 年光伏产业链:
- 硅片产能: 500 GW (vs 需求 350 GW) +43% 过剩
- 隆基降价: 3 元/W → 1.5 元/W (-50%)
- 毛利率: 25% → 12% (-13pct)

为了抢市场:
1. 低价赊销 (应收账款暴增)
2. 生产堆库存 (存货暴增)
3. 账面盈利, 现金流血

结果: 2024 年计提减值 80 亿 ❌
```

## 实战决策

### ✅ 立即卖出信号
1. **现金流转化率 < 0%** - 严重失血
2. **应收账款暴增 +80%** - 大量坏账风险
3. **存货积压 +60%** - 产品滞销
4. **资产负债率 > 60%** - 债务压力大

### ⚠️ 即将爆雷
1. **3 个月内计提减值** - 应收 + 存货减值 80 亿
2. **股价暴跌 -43%** - 市场提前反应
3. **券商下调评级** - "买入" → "卖出"
4. **融资困难** - 银行不再放贷

### 🎯 操作策略

**策略 A: 立即清仓 (唯一正确选择)**
```
看到 2023 Q3 财报:
- 发现现金流 -50 亿 ❌
- 应收账款 +80%, 存货 +60% ⚠️
- 立即卖出 @ 35 元

3 个月后 (2024 Q1):
- 股价 20 元 (-43%)
- 避免损失 -43% ✅

如果不卖:
- 35 元 → 20 元, 损失 -15 元/股
- 100 万投资 → 57 万 ❌
```

**策略 B: 抄底赌反转? (危险) ❌**
```
有人想: 股价跌 -43%, 便宜了, 抄底?

风险:
1. 行业持续恶化 (光伏产能过剩 +50%)
2. 财务继续恶化 (现金流可能更差)
3. 可能继续跌 -50%+

历史案例:
- 东旭光电: 财务危机 → 股价跌 -90%
- 乐视网: 现金流危机 → 退市

教训: 财务危机不要抄底!
```
