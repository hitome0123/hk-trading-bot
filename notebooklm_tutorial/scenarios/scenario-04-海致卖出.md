## 核心知识点

### 1. 三表关系

```
利润表 (Income Statement)
├─ 营收
├─ 成本
└─ 净利润 ← 这是"账面利润"

现金流量表 (Cash Flow)
├─ 经营现金流
├─ 投资现金流
└─ 筹资现金流 ← 这是"真金白银"

资产负债表 (Balance Sheet)
├─ 资产 (现金、应收账款、存货)
├─ 负债 (应付账款、借款)
└─ 权益 ← 净资产
```

**关键逻辑**: 净利润高，但现金流低 → 盈利质量差

---

### 2. 盈利质量检查清单

| 指标 | 健康标准 | 海致案例 | 评分 |
|------|---------|---------|------|
| **经营现金流/净利润** | > 100% | 67% | ❌ |
| **应收账款/营收** | < 30% | 45% | ⚠️ |
| **存货周转天数** | 下降 | 上升 | ❌ |
| **自由现金流** | > 净利润 | < 净利润 | ❌ |
| **负债率** | < 50% | 38% | ✅ |

**结论**: 海致盈利质量一般（5 项中 1 项优秀，3 项不达标）

---

### 3. 卖出信号

**何时该卖？**
1. ❌ 盈利质量恶化（经营现金流/净利润 < 70%）
2. ❌ 应收账款激增（坏账风险）
3. ❌ 存货积压（产品滞销）
4. ✅ 估值过高（PE > 50x且无增长）
5. ✅ 达到目标价（+50% 已达成）

**海致案例**: 满足 1-3 项卖出信号 → 建议部分获利

## 实战决策

### 🎯 操作策略

**方案 A: 分批获利 (推荐)**
```
卖出 50% @ HK$120 (锁定 +50% 收益)
保留 50% (赌业绩持续改善)
剩余仓位止损: HK$100 (-17%)
```

**方案 B: 全部卖出 (保守)**
```
盈利质量差 + 涨幅达标 → 全部卖出
等待回调至 HK$90-100 再买回
```

**方案 C: 继续持有 (激进)**
```
如果看好行业 + 相信管理层能改善现金流
设置移动止损: 从最高点回撤 15% 时卖出
```
