## 核心知识点

### 1. 周期股估值陷阱

**什么是周期股**:
```
周期股 = 盈利随经济/行业周期波动的股票
动力电池 = 典型周期股 (跟随新能源车销量)
```

**周期股估值陷阱: PE 低 ≠ 便宜**
```python
# 案例: 宁德时代
2021 年 (周期顶部):
- 营收: 1,300 亿, 净利润: 160 亿
- PE: 60x, 股价: 600 元
- 看似贵, 但其实是卖点 ⚠️

2023 年 (周期底部):
- 营收: 800 亿 (-38%), 净利润: 40 亿 (-75%)
- PE: 150x, 股价: 300 元
- 看似便宜, 其实是最贵的时候 ❌

教训: 周期股看 PE 会踩雷, 要用 PB 或 PS
```

**正确估值方法: PB (市净率)**
```python
PB = 市值 / 净资产

宁德时代:
- 市值: 1.3 万亿
- 净资产: 2,000 亿
- PB = 6.5x

行业对比:
- 比亚迪: PB 5.0x
- 亿纬锂能: PB 4.5x
- 行业中位数: PB 5.0x

结论: 宁德 PB 6.5x, 溢价 30% ⚠️
```

---

### 2. 产能过剩信号

**行业产能 vs 需求**:
```python
# 2024 年数据
全球动力电池产能: 1,200 GWh
全球电动车需求: 800 GWh (假设电池需求 100 kWh/车, 800 万辆)
产能利用率: 800 / 1,200 = 67% ⚠️

健康标准: 产能利用率 > 80%
过剩标准: 产能利用率 < 70% ⚠️

2024 年产能过剩 33% → 价格战不可避免
```

**价格战的影响**:
```python
宁德时代碳酸锂电池价格:
2022 年: 150 元/kWh (高点)
2023 年: 100 元/kWh (-33%)
2024 年: 80 元/kWh (-20%, 继续下跌)

毛利率下滑:
2022 年: 32%
2023 年: 28% (-4pct)
2024E: 25% (-3pct)

→ 利润率持续压缩 ⚠️
```

---

### 3. 市占率陷阱

**宁德时代市占率变化**:
```
2021 年: 50% (绝对龙头)
2022 年: 40% (比亚迪崛起)
2023 年: 35% (继续下滑)
2024E: 32% (竞争加剧)

市占率下滑原因:
1. 比亚迪自产自销 (刀片电池)
2. 二线厂商 (亿纬, 国轩) 低价抢单
3. 海外厂商 (LG, 松下) 复苏
```

**市占率 vs 估值**:
```python
市占率下滑 → 定价权丧失 → 利润率下降 → 估值下调

宁德合理 PE:
- 市占率 50% 时: PE 80x (龙头溢价)
- 市占率 35% 时: PE 50x (竞争加剧)
- 市占率 30% 时: PE 40x (失去定价权)

当前 PE 60x vs 合理 50x → 溢价 20% ⚠️
```

### 4. 对比美股: 特斯拉 vs 宁德时代

**特斯拉电池 (4680)**:

策略: 自产电池 (不依赖宁德)
成本: 降低 50% (vs 外购宁德电池)

对宁德的影响:
- 失去最大客户 (特斯拉占宁德 15% 收入)
- 营收增速放缓 +30% → +15%
```

**估值对比**:
```
特斯拉 (2024):
- PE: 60x (vs 宁德 60x)
- 市占率: 20% (全球电动车)
- 增速: +25%

宁德时代:
- PE: 60x
- 市占率: 35% (全球动力电池)
- 增速: +30%

相同 PE, 但:
- 特斯拉有自动驾驶, AI 故事
- 宁德只有电池, 单一业务

结论: 宁德应该比特斯拉便宜 (PE 40-50x)

## 实战决策

### ✅ 继续持有理由
1. **龙头地位稳固** - 市占率 35%, 全球第一
2. **技术护城河** - 钠离子电池, 固态电池研发领先
3. **长期赛道** - 电动车渗透率 30% → 70% (10 年)
4. **估值不算离谱** - PE 60x, PEG 2.0 (可接受)

### ⚠️ 减仓理由
1. **产能过剩 33%** - 价格战不可避免
2. **利润率下滑** - 毛利率 32% → 25%
3. **市占率下降** - 50% → 35% (定价权丧失)
4. **估值偏高** - PE 60x, 溢价 20% vs 合理值 50x

### 🎯 操作策略

**推荐策略: 分层减持**
```
立即卖出:
- 小票 (国轩, 欣旺达) 全部清仓
- 锁定利润, 避免泡沫破裂

保留龙头:
- 宁德时代保留 50% (减持 50%)
- 比亚迪保留 100% (综合性更强)
- 设止损 -15% (从当前价)

观察信号:
- 如果板块 PE > 80x → 全面撤退
- 如果产能利用率 < 60% → 龙头也减半
- 如果毛利率 < 20% → 行业进入寒冬
```

### 实战案例: 光伏板块教训

**光伏板块 (2020-2022)**:
```
2020 年疯涨:
- 隆基绿能: PE 60x → 120x (+100%)
- 通威股份: PE 40x → 80x (+100%)
- 产能利用率: 90% (供不应求)

2022 年崩盘:
- 隆基绿能: PE 120x → 15x (-87%)
- 通威股份: PE 80x → 12x (-85%)
- 产能利用率: 50% (产能过剩)

教训:
1. 产能过剩 = 行业拐点
2. 周期股不能用 PE 估值
3. 高增长不可持续, 迟早回归周期
```

**对宁德的启示**:
- 动力电池 = 第二个光伏 (同样产能过剩)
- PE 60x 已经很高, 不要期待继续上涨
- 及时减仓, 保住利润
