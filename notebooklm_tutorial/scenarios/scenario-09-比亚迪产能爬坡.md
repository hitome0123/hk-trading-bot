## 核心知识点

### 1. 产能爬坡的四个阶段

**阶段 1: 宣布投产 (T-30天)**
```
消息发布: 官宣新工厂投产计划
市场反应: 温和上涨 +5-10% (炒作预期)
投资者行为: 聪明资金开始建仓

案例: 比亚迪 3 月初官宣 → 股价 +8%
```

**阶段 2: 试产启动 (T)**
```
工厂状态: 小批量试产, 验证产线
市场反应: 加速上涨 +10-15% (预期强化)
投资者行为: 散户跟进, 成交量放大

案例: 西安工厂 2023-02 试产 → 股价 +12%
```

**阶段 3: 爬坡期 (T+1 ~ T+3月)**
```
工厂状态: 产能逐步提升 30% → 60% → 90%
市场反应: 震荡上行 +15-25% (验证进度)
投资者行为: 博弈爬坡速度, 波动加大

案例: 西安工厂 2-5 月 → 股价 +28%
```

**阶段 4: 满产兑现 (T+3月)**
```
工厂状态: 满产达标, 销量数据验证
市场反应: 冲高回落 (利好出尽)
投资者行为: 获利了结, 成交量萎缩

案例: 西安工厂 5 月满产 → 股价见顶后回调 -8%
```

---

### 2. 产能催化剂的三大指标

**指标 1: 产能利用率**
```python
产能利用率 = 实际产量 / 设计产能

比亚迪郑州工厂:
- 设计产能: 150 万辆/年 = 12.5 万辆/月
- 4 月实际: 4 万辆/月 → 利用率 32% ⚠️ 爬坡初期
- 5 月实际: 8 万辆/月 → 利用率 64% ✅ 爬坡加速
- 6 月实际: 12 万辆/月 → 利用率 96% ✅ 接近满产

满产标准: 利用率 > 90% ✅
```

**指标 2: 单车成本下降**
```python
规模效应 = 固定成本分摊降低

比亚迪单车成本:
- 爬坡前 (产能 100 万): 15 万元/辆
- 满产后 (产能 150 万): 14.25 万元/辆
- 成本下降: 5%

利润弹性 = 5% × 销量 150 万 = +112.5 亿利润
→ 净利润增长 +15% ✅
```

**指标 3: 良率爬坡速度**
```python
良率 = 合格品数量 / 总产量

比亚迪历史良率爬坡:
- 第 1 个月: 70% (试产调试)
- 第 2 个月: 85% (稳定期)
- 第 3 个月: 95% (成熟期)

良率 > 95% = 满产标准 ✅
```

### 3. 历史案例: 特斯拉上海工厂

**特斯拉上海超级工厂 (2019-2020)**:
```
2019-10: 官宣投产
2020-01: 开始交付 (产能 15 万辆/年)
2020-06: 满产 (产能 50 万辆/年)
2020-12: 扩产 (产能 75 万辆/年)

股价表现:
- 投产前 (2019-10): $250
- 满产时 (2020-06): $180 (-28%) ⚠️ COVID-19 影响
- 扩产后 (2020-12): $700 (+180%) ✅

教训:
1. 宏观环境 > 产能催化 (COVID-19 压制)
2. 长期持有赚钱 (2 年 +180%)
3. 产能扩张不是单次催化, 是持续利好
```

### 4. 风险识别: 爬坡失败案例

**蔚来合肥工厂 (2021-2022)**:
```
2021-10: 官宣合肥二期投产 (产能 +10 万辆)
2022-01: 试产启动
2022-04: 爬坡受阻 (疫情 + 供应链断裂)
2022-07: 产能利用率仅 40% ⚠️ 远低于预期

股价表现:
- 投产前 (2021-10): $40
- 爬坡期 (2022-01-07): $30-20 (-25% ~ -50%)

失败原因:
1. 疫情导致供应链中断 (芯片短缺)
2. 良率低 (新产线调试问题)
3. 需求不足 (市场竞争加剧)

教训:
- 不是所有产能扩张都成功
- 必须验证: 供应链 + 需求 + 良率
```

## 实战决策

### ✅ 买入信号
1. **官宣投产计划** - 时间表明确, 资金到位
2. **历史成功案例** - 同公司过往爬坡成功率 > 80%
3. **需求旺盛** - 订单积压, 产销两旺
4. **供应链稳定** - 关键零部件供应无风险

### ⚠️ 风险提示
1. **爬坡速度不及预期** - 良率问题, 设备故障
2. **需求下滑** - 市场竞争, 补贴退坡
3. **宏观环境恶化** - 疫情, 经济衰退
4. **估值过高** - PE 60x, 已提前透支产能利好

### 🎯 操作策略

**策略 A: 标准催化剂交易 (推荐)**
```
买入时机: 投产前 1-2 周 (3/15-3/25)
买入价: 200 元 (当前价)
持仓期: 3-4 个月 (至满产)
卖出时机: 满产前 1 周 (6/20) 或满产当天
目标收益: +25-35%
止损: -10% (如果爬坡失败)
```

**策略 B: 分批建仓 (保守)**
```
第一批: 投产前 1 周买入 30% @ 200 元
第二批: 试产启动买入 30% @ 210 元
第三批: 爬坡加速买入 40% @ 220 元

优点: 降低单次押注风险
缺点: 平均成本较高
```

**策略 C: 长期持有忽略催化剂 (长线)**
```
如果是长期投资者:
- 产能扩张只是阶段性利好
- 继续持有, 享受成长红利
- 满产后股价可能震荡, 但长期向上

适合: 比亚迪基本面信仰者
```
