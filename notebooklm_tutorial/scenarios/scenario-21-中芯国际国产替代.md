## 核心知识点

### 1. TAM 三层金字塔

**Level 1: TAM (Total Addressable Market)**
```
定义: 理论上最大的可触达市场
中芯 TAM = 中国半导体市场 = $250B/年

计算方法:
- 中国半导体进口: $400B/年
- 中国本土生产: $100B/年
- 减去不可替代部分 (先进制程): -$250B
- TAM = $250B ✅
```

**Level 2: SAM (Serviceable Available Market)**
```
定义: 企业有能力服务的市场
中芯 SAM = 成熟制程市场 (28nm/14nm) = $150B/年

限制因素:
- 技术能力: 中芯只能做 28nm/14nm, 不能做 5nm/3nm
- 生产能力: 产能 600 万片/年 (vs 台积电 1,200 万片)
- SAM = TAM × 60% = $150B
```

**Level 3: SOM (Serviceable Obtainable Market)**
```
定义: 企业实际能抢到的市场份额
中芯 SOM = SAM × 国产替代率 50% = $75B/年

影响因素:
- 竞争: 台积电, 三星, 格罗方德
- 客户粘性: 华为, 小米愿意用国产芯片
- 价格: 国产芯片价格 -30% vs 台积电
- SOM = $75B (乐观假设)
```

### 2. 国产替代率的合理性验证

**当前国产化率 (2024)**:
```python
# 各细分领域国产化率
存储芯片 (DRAM/NAND): 5% (长江存储, 长鑫存储)
逻辑芯片 (28nm): 15% (中芯国际)
模拟芯片: 25% (卓胜微, 韦尔股份)
功率芯片: 30% (闻泰科技)

平均国产化率: 18.75%
```

**5 年后目标 (2029)**:
```python
# 政策目标: 《中国制造 2025》
2025 年国产化率目标: 70%
实际可达: 50% (保守估计) ✅

细分领域:
存储芯片: 15% (vs 目标 30%)
逻辑芯片: 50% (vs 目标 70%) ← 中芯主要贡献
模拟芯片: 60% (vs 目标 80%)
功率芯片: 70% (vs 目标 90%)

加权平均: 48.75% ≈ 50% ✅ 合理
```

**验证方法: 对比韩国历史**:
```
韩国半导体国产化 (1980-2000):
- 1980 年: 5% (完全依赖美国)
- 1990 年: 30% (三星, SK 海力士崛起)
- 2000 年: 70% (全球第二大芯片生产国)

用时: 20 年, 从 5% → 70%
年均提升: 3.25 个百分点

中国路径 (2020-2030):
- 2020 年: 15%
- 2025 年: 35% (+4 pct/年)
- 2030 年: 50% (+3 pct/年)

结论: 50% 是可达成的保守目标 ✅
```

### 3. 市值天花板估算

**方法 1: P/S (市销率) 估值**
```python
# 假设国产替代 50%
营收 = $75B × 70% (中芯市占率) = $52.5B (367 亿元)

P/S 倍数 (参考台积电):
- 台积电 P/S: 8-10x (成熟稳定)
- 中芯 P/S: 10-12x (成长溢价)

市值 = 367 亿 × 12 = 4,404 亿元
vs 当前 3,000 亿 → +47% ✅
```

**方法 2: 产能 × 单价估值**
```python
# 中芯产能规划 (2029)
产能: 1,200 万片 12 寸晶圆/年 (vs 当前 600 万片)

平均售价:
28nm: $2,000/片
14nm: $3,500/片
加权平均: $2,500/片

营收 = 1,200 万片 × $2,500 = $300 亿 (2,100 亿元)
市值 = 2,100 亿 × 3 (P/S) = 6,300 亿元 ✅

vs 当前 3,000 亿 → +110% ✅
```

### 4. 风险因素分析

**风险 1: 技术突破不及预期**
```
关键节点:
- 2025 年: 14nm 量产 (目标 50 万片/月)
- 2027 年: 7nm 量产 (目标 30 万片/月)

如果延迟:
- SAM 缩小 -30% ($150B → $105B)
- 市值天花板下降 -30% (6,300 亿 → 4,400 亿)
```

**风险 2: 美国进一步制裁**
```
最坏情况:
- EUV 设备禁运 (无法生产 7nm 以下)
- 现有设备禁止维护 (产能下降 -50%)
- 客户流失 (华为, 小米 被迫用台积电)

影响: 市值下跌 -50% (3,000 亿 → 1,500 亿) ❌
```

**风险 3: 需求不足**
```
中国手机市场萎缩:
- 2023 年: 2.8 亿部 (-5%)
- 2024 年: 2.6 亿部 (-7%)
- 需求下降 → 芯片订单减少

影响: 营收增速 +20% → +10%, 估值下调 -20%
```

## 实战决策

### ✅ 买入理由
1. **TAM 巨大** - 中国半导体市场 $250B, 增长空间 3-5x
2. **政策支持** - 国产替代, 大基金二期 3,000 亿投资
3. **估值低** - 当前 3,000 亿 vs 合理 6,000 亿 (折价 50%)
4. **技术进步** - 14nm 已量产, 7nm 2027 年量产

### ⚠️ 风险提示
1. **技术不确定性** - 7nm 量产可能延迟
2. **美国制裁风险** - EUV 设备禁运
3. **周期性** - 半导体行业周期波动大
4. **估值不便宜** - P/S 10x, 高于行业 8x

```
- 2029 年目标价: 80 元 (+100%)

```

```

- 现在买入 @ 40 元

```
