## 核心知识点

### 1. 分红的三个核心指标

**指标 1: 分红率 (Payout Ratio)**
```python
分红率 = 分红总额 / 净利润

贵州茅台:
分红率 = 339 亿 / 606 亿 = 56%

评级标准:
- > 60%: 慷慨分红 (现金奶牛)
- 40-60%: 健康分红 (平衡成长和分红) ✅ 茅台
- 20-40%: 低分红 (保留资金扩张)
- < 20%: 几乎不分红 (成长股)
```

**指标 2: 股息率 (Dividend Yield)**
```python
股息率 = 每股分红 / 股价

贵州茅台:
股息率 = 270 / 1800 = 1.5%

对比其他投资:
- 余额宝: 2.5% (无风险)
- 10 年期国债: 2.8%
- 银行理财: 3.0-4.0%
- 茅台股息: 1.5% ⚠️ 低于无风险利率

结论: 茅台不是纯分红股, 主要看资本利得 (股价上涨)
```

**指标 3: 填权概率**
```python
填权 = 除权后股价回到除权前水平

贵州茅台历史:
- 2022 年: 除权后 15 天, 填权 100%
- 2021 年: 除权后 10 天, 填权 100%
- 2020 年: 除权后 20 天, 填权 80%

茅台填权成功率: 100% (5/5) ✅
平均填权时间: 15 天
```

---

### 2. 除权除息机制 (A股特有)

**什么是除权除息**:
```
除权除息日: 6/30
股权登记日: 6/29

6/29 收盘持有 → 获得分红 270 元
6/30 开盘股价调整 → 1800 - 270 = 1530 元
```

**为什么要除权**:
```
目的: 防止分红套利
如果不除权:
- 投资者 A: 6/29 买入 @ 1800 元
- 6/30 获得分红 270 元
- 立即卖出 @ 1800 元
- 套利 270 元 (无风险) ⚠️

除权后:
- 6/29 买入 @ 1800 元
- 6/30 股价 1530 元 + 分红 270 元 = 1800 元
- 总资产不变 ✅
```

---

### 3. 填权与贴权

**填权 (股价回升)**:
```
除权后股价: 1530 元
15 天后: 1780 元 (+16.3%)
填权幅度: 250 / 270 = 93% ✅

投资者总收益:
- 分红: 270 元
- 填权: 250 元
- 合计: 520 元 (+28.9%)
```

**贴权 (股价继续跌)**:
```
除权后股价: 1530 元
30 天后: 1450 元 (-5.2%) ⚠️
贴权幅度: -80 元

投资者总收益:
- 分红: 270 元
- 贴权: -350 元 (1800→1450)
- 合计: -80 元 (-4.4%) ❌
```

**填权 vs 贴权的决定因素**:
1. **基本面**: 公司业绩持续增长 → 填权
2. **市场情绪**: 牛市 → 填权, 熊市 → 贴权
3. **估值**: 除权后估值合理 → 填权

---

### 4. 分红陷阱: 高分红不等于好股票

**陷阱案例: 某钢铁股**
```
2023 年分红:
- 每股分红: 1.2 元
- 股价: 6 元
- 股息率: 20% ⚠️ 远超银行理财

看似诱人, 但:
- 分红后股价: 4.8 元 (除权 -20%)
- 1 年后股价: 3.5 元 (贴权 -27%)
- 总收益: 1.2 - 2.5 = -1.3 元 (-21.7%) ❌

陷阱原因:
1. 行业衰退 (钢铁产能过剩)
2. 无成长性 (净利润连续 3 年下滑)
3. 一次性分红 (变卖资产凑钱分红)
```

**识别分红陷阱**:
```python
# 检查指标
1. 分红率 > 100% → 警告 (分红超过利润, 不可持续)
2. 净利润增速 < 0 → 警告 (业绩下滑)
3. 历史分红不稳定 → 警告 (一次性分红)
4. 股息率 > 10% → 警告 (可能是股价暴跌导致)
```

## 实战决策

### ✅ 买入信号 (吃分红)
1. **高分红 + 高成长** - 茅台分红率 56%, 净利润增长 15%
2. **历史填权成功** - 过去 5 年填权率 100%
3. **估值合理** - PE 35x, 低于历史中位数 40x
4. **牛市环境** - 市场情绪好, 填权概率高

### ⚠️ 风险提示
1. **股息率低** - 1.5% 低于无风险利率 2.8%
2. **贴权风险** - 如果熊市, 可能不填权
3. **机会成本** - 分红后资金闲置, 错过其他机会
4. **估值风险** - PE 35x 已不便宜

### 🎯 操作策略

**策略 A: 长期持有吃分红 (推荐)**
```
适合: 茅台基本面信仰者

操作:
1. 6/29 前持有 (获得分红 270 元)
2. 除权后继续持有 (等待填权)
3. 填权完成后 (1780 元) 可选择卖出

预期收益:
- 分红: 270 元
- 填权: 250 元
- 合计: +28.9%
```

**策略 B: 除权后买入 (填权套利)**
```
适合: 短线交易者

操作:
1. 6/30 除权后买入 @ 1530 元
2. 等待 15 天填权 → 1780 元
3. 填权完成立即卖出

预期收益:
- 填权: +16.3%
- 时间: 15 天
- 年化收益: +400%

风险: 如果不填权, 可能亏损
```

**策略 C: 除权前卖出 (规避风险)**
```
适合: 不看好茅台 + 追求确定性

操作:
1. 6/29 前卖出 @ 1800 元
2. 避免除权下跌 270 元
3. 观望填权行情

缺点:
- 错过分红 270 元
- 错过填权机会
```
