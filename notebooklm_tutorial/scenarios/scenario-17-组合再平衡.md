- 科技股 40% ($40K)

- 现金 10% ($10K)

12 个月后 (2024 年底):
- 科技股涨 80% → $72K (占比 57%) ⚠️
- 消费股涨 10% → $33K (占比 26%)
- 金融股跌 5% → $19K (占比 15%)
- 现金不变 → $2K (占比 2%, 红利再投资)

总资产: $126K (+26%)

问题: 科技股占比从 40% 飙升至 57%, 风险敞口过大!

你该卖掉部分科技股, 还是继续持有享受涨势?

## 核心知识点

### 1. 再平衡的两大作用

**作用 1: 风险控制**
```
科技股占比 57% (vs 目标 40%):
- 如果科技股跌 30%
- 组合损失 = 57% × (-30%) = -17.1%

再平衡后科技股 40%:
- 组合损失 = 40% × (-30%) = -12%
- 少亏 5.1%! ✅
```

**作用 2: 强制"高抛低吸"**
```
再平衡 = 卖出涨多的, 买入跌多的
- 卖科技股 @ 高位
- 买金融股 @ 低位
- 自动实现"逆向投资"
```

---

### 2. 再平衡触发条件

**触发规则**:
```
偏离度 > 15% → 必须再平衡
偏离度 10-15% → 建议再平衡
偏离度 < 10% → 可不再平衡

科技股偏离 +17% → 触发必须再平衡 ⚠️
```

**再平衡频率**:
- 季度再平衡: 最优 (回测夏普比率 0.90)
- 年度再平衡: 次优 (夏普 0.85)
- 月度再平衡: 过度交易 (手续费侵蚀收益)

---

### 3. 再平衡成本计算

**成本 1: 资本利得税**
```python
卖出 NVDA $21K (成本 $10K)
利润 = $21K - $10K = $11K
税 = $11K × 15% = $1,650

卖出 MSFT $10K (成本 $8K)
利润 = $2K
税 = $2K × 15% = $300

总税 = $1,950
```

**成本 2: 交易手续费**
```
卖出 2 笔 + 买入 2 笔 = 4 笔
手续费 $5/笔 × 4 = $20 (可忽略)
```

**成本 3: 价差损失**
```
买卖价差 0.1%
$21K × 0.1% = $21

总成本 = $1,950 + $20 + $21 = $1,991
```

---

### 4. 税务优化再平衡

**策略: 用新资金再平衡 (避免卖股)**
```
不卖科技股, 用新投入资金买其他股:

每月新投入 $2K × 6 个月 = $12K
- 买入消费股 $5K
- 买入金融股 $6K
- 现金 $1K

6 个月后:
- 科技 $72K → 占比 52% (vs 原 57%)
- 消费 $38K → 占比 28%
- 金融 $25K → 占比 18%
- 偏离度降至 12% (vs 原 17%)

优点: 零税成本! ✅
缺点: 需要 6 个月才能完成
```

## 实战决策

### ✅ 立即再平衡理由
1. **偏离度 +17%** - 触发强制再平衡线
2. **科技股估值贵** - PE 60x, 回调风险大
3. **风险过度集中** - 57% 单一板块
4. **历史规律** - 年度再平衡跑赢不平衡

### ⚠️ 不再平衡理由
1. **科技股趋势强** - AI 革命刚开始
2. **资本利得税高** - 触发 $1,950 税
3. **金融股基本面弱** - 利率下降不利银行
4. **让赢家奔跑** - 巴菲特策略

### 🎯 操作策略

**推荐策略: 方案 B (部分再平衡)**
```
立即调整:
- 卖出科技股 $10K (仅卖一半)
- 买入金融股 $6K
- 保留现金 $4K

调整后配置:
- 科技 49% (vs 目标 40%, 偏离 +9%) ✅ 可接受
- 消费 26% (vs 目标 30%, 偏离 -4%) ✅ 可接受
- 金融 20% (vs 目标 20%, 偏离 0%) ✅ 达标
- 现金 5% (vs 目标 10%, 偏离 -5%) ⚠️ 略低

优点:
- 降低风险敞口
- 税成本减半 ($975 vs $1,950)
- 仍保留科技股上涨潜力

后续调整:
- 3 个月后用新资金继续平衡
```
