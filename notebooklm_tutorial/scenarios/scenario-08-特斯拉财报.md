## 核心知识点

### 1. 美股盘后交易 vs 港股盘后差异

**美股盘后交易特点**:
| 特征 | 美股 | 港股 |
|------|------|------|
| 时间 | 16:00-20:00 ET (4小时) | 17:00-19:00 HKT (2小时) |
| 流动性 | 盘中的 15-25% | 盘中的 5-10% |
| 价差 | 1-3% (蓝筹), 3-5% (小票) | 2-3% (蓝筹), 5%+ (小票) |
| 操纵风险 | 中等 | 高 |
| 散户参与 | 30-40% | 10-20% |

**特斯拉盘后特点**:
- 流动性相对较好 (日均成交额 $80 亿)
- 盘后成交量约 2000 万股 (vs 盘中 1.2 亿股)
- 价差通常 1-2% (蓝筹级别)

---

### 2. 财报超预期 3 大维度

**维度 1: EPS Beat**
```python
EPS Beat % = (实际 EPS - 预期 EPS) / 预期 EPS
特斯拉: (1.02 - 0.85) / 0.85 = 20% ✅ (显著超预期)

Beat 幅度判断:
- < 5%: 轻微超预期 (股价 +2-3%)
- 5-10%: 中等超预期 (股价 +5-8%)
- 10-20%: 显著超预期 (股价 +10-15%)
- > 20%: 爆表超预期 (股价 +15%+)
```

**维度 2: Revenue Beat**
```python
Revenue Beat % = (实际营收 - 预期营收) / 预期营收
特斯拉: (290 - 275) / 275 = 5.5% ✅ (中等超预期)
```

**维度 3: Guidance**
- 上调 (Raised) = 强烈看好 → 股价 +5-10%
- 维持 (Maintained) = 中性 → 股价 ±2%
- 下调 (Lowered) = 利空 → 股价 -5-15%

**特斯拉案例: 三重超预期 (EPS + Revenue + Guidance) → 股价应涨 15-20%**

---

### 3. "利好出尽即利空" vs "持续上涨"

**两种财报后走势**:

**模式 A: 利好出尽 (60% 概率)**
```
财报前 → 预期发酵 → 股价提前上涨 +10%
财报后 → 利好落地 → 获利盘了结 → 股价回落 -5%
净效果: +5%
```

**模式 B: 持续上涨 (40% 概率)**
```
财报后 → 超预期太强 → FOMO 情绪 → 继续上涨 +10%
条件: EPS beat > 15% + 指引上调 + 板块轮动
```

**如何判断**:
1. **财报前涨幅**: 如果已涨 > 10%, 多为模式 A
2. **超预期幅度**: EPS beat > 20%, 多为模式 B
3. **板块情绪**: 电动车板块整体火热, 多为模式 B
4. **技术面**: 突破前高 + 放量, 多为模式 B

**特斯拉案例判断**:
- 财报前涨幅: +5% (不算多) ✅
- EPS beat: 20% (爆表) ✅
- 板块情绪: 比亚迪/理想汽车同步上涨 ✅
- **预测**: 70% 概率模式 B (持续上涨)

### 4. 美股盘后交易策略

**策略 A: 不参与盘后 (推荐散户)**
```
原因:
- 流动性差 20% → 价差大 1-3%
- 容易被操纵 → 假突破
- 情绪化交易 → 追高风险

操作:
- 等开盘后 30 分钟观察
- 如果回落至 +5-7%, 买入
- 如果继续冲高 > +12%, 不追
```

**策略 B: 小仓位试探 (适合老手)**
```
盘后买入 10% 仓位 @ $280
开盘后:
- 涨幅持续: 加仓至 30%
- 涨幅回落: 止损出局 -2%
```

**策略 C: 期权对冲 (专业)**
```
盘后买入股票 100 股 @ $280
同时买入 put option (行权价 $275, 到期 1 周)
对冲盘后波动风险
```

## 实战决策

### ✅ 财报质量高
1. **三重超预期** - EPS 20% + Revenue 5.5% + Guidance up
2. **毛利率提升** - 17.8% → 19.5% (盈利质量改善)
3. **指引上调** - Q1 交付量 +10% (重大拐点)
4. **竞争力增强** - 成本下降, 价格战中占优

### ⚠️ 追高风险
1. **盘后涨幅大** - 12% 已透支 50-70% 利好
2. **流动性差** - 盘后价差 1-2%, 成本高
3. **竞争加剧** - 比亚迪全球销量超越 (2024 Q4)
4. **估值偏贵** - PE 65x, 高于行业平均 40x

### 🎯 操作策略

**策略 A: 不追高，等回调 (推荐)**
```
开盘观察涨幅:
- 如果 < +7%: 买入 (利好未充分反映)
- 如果 +7-12%: 观望 (合理涨幅)
- 如果 > +12%: 不买 (过度反应)

买入价: $265-275
目标价: $300 (+13-20%)
止损价: $240 (-8%)
```

**策略 B: 盘后小仓位 (激进)**
```
盘后买入 10% 仓位 @ $280
开盘后:
- 涨幅 > +15%: 加仓至 30%
- 涨幅 +10-15%: 持有
- 涨幅 < +10%: 止损 @ $275
```
