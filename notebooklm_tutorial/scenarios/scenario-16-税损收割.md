## 学习目标

## 背景故事

2024 年你的美股组合:
- 盈利股: NVDA +80% ($50K 利润)
- 亏损股: PLTR -30% ($15K 亏损)
- 总盈利: $35K

如果不做任何操作:
- 资本利得税 (长期): 15% × $35K = $5,250
- 税后净利润: $29,750

你的会计师建议: 使用 Tax Loss Harvesting 可以省税 $2,250!

但你担心: 卖掉 PLTR 后, 30 天内不能买回 (Wash Sale 规则)。如果 PLTR 这 30 天暴涨怎么办?

## 核心知识点

### 1. 美股资本利得税规则

**税率**:
```
短期资本利得 (持有 < 1 年):
- 按普通收入税率 (10%-37%)

长期资本利得 (持有 > 1 年):
- 0% (收入 < $44K)
- 15% (收入 $44K-$492K) ← 大多数人
- 20% (收入 > $492K)
```

**抵扣规则**:
```
亏损可抵扣盈利:
- 当年亏损 $15K, 盈利 $50K
→ 净盈利 $35K, 税 $5,250

亏损超过盈利:
- 当年亏损 $60K, 盈利 $50K
→ 净亏损 $10K
→ 可抵扣当年普通收入 $3K
→ 剩余 $7K 结转未来 5 年
```

---

### 2. Wash Sale 规则 (30 天禁买)

**核心规则**:
```
卖出亏损股后 30 天内买回 = Wash Sale
→ 亏损不能抵扣当年税
→ 亏损加入新买入股票的成本

例子:
12/15 卖出 PLTR @ $20 (成本 $30, 亏损 $10)
12/20 买回 PLTR @ $22
→ 亏损 $10 无效
→ 新成本 = $22 + $10 = $32
```

**如何规避**:
1. 等 31 天后再买回
2. 买入"实质不同"的替代股 (SNOW vs PLTR)
3. 让配偶买入 (不算 Wash Sale)

---

### 3. 税损收割最佳时机

**12 月策略**:
```
12/1-12/15: 最佳收割期
- 距离年底还有 2 周
- 有时间重新买入 (30 天后)

12/16-12/31: 紧急收割期
- 如果错过前面, 抓紧最后机会
- 但无法在当年重新买入
```

**全年策略**:
```
任何时候亏损 > 20%:
- 考虑立即收割
- 锁定亏损, 抵扣未来盈利
```

---

### 4. 高级策略: 对冲收割

**策略**:
```
12/15 卖出 PLTR 100 股 @ $20 (亏损 $1K)
12/15 买入 PLTR 看涨期权 (行权价 $20, 到期 1/30)

效果:
- 锁定税损 $1K
- 如果 PLTR 涨, 期权赚钱
- 30 天后行权, 重新持有股票

成本:
- 期权费 $200-300
- 但省税 $150 (15% × $1K)
```

## 实战决策

### ✅ 适合税损收割的股票
1. **亏损 > 15%** - 税务优惠显著
2. **不看好长期** - 卖了也不想买回
3. **有替代股** - 可规避 Wash Sale
4. **年底前发现** - 12 月 1-15 日

### ⚠️ 不适合收割的股票
1. **亏损 < 10%** - 税务优惠不明显
2. **长期看好** - 卖了可能错过反弹
3. **无替代股** - 30 天空窗期风险大
4. **已接近年底** - 12 月 20 日后

### 🎯 操作策略

**推荐策略: 方案 B (部分收割)**
```
12/10 卖出 PLTR 全部仓位
- 锁定亏损 $15K
- 可抵扣未来 5 年盈利
- 省税 $2,250 (15% × $15K)

12/10-1/10 (30 天):
- 观察 PLTR 走势
- 如果暴跌 → 不买回 (避免接飞刀)
- 如果横盘 → 1/10 买回
- 如果暴涨 → 买入替代股 SNOW

风险控制:
- 如果 PLTR 30 天内涨 > 15% → 损失机会成本
- 但省税 $2,250 仍然值得
```
