## 核心知识点

### 1. 财报超预期识别

**三大超预期指标**:
1. **EPS Beat**: (实际 EPS - 预期 EPS) / 预期 EPS
   - 智谱案例: (-0.22 - (-0.35)) / -0.35 = 37% 超预期 ✅

2. **Revenue Beat**: (实际营收 - 预期营收) / 预期营收
   - 智谱案例: (18 - 15) / 15 = 20% 超预期 ✅

3. **Guidance**: 管理层对下季度的指引
   - 上调 = 强烈看好
   - 维持 = 中性
   - 下调 = 利空

---

### 2. 盘后交易风险

**港股盘后交易特点**:
- 时间: 17:00 - 19:00（2 小时）
- 流动性: 仅为盘中的 5-10%
- 价差: 买卖价差宽 (可达 2-3%)
- 风险: 容易被大单操纵

**是否参与盘后交易？**
- ❌ 不建议: 散户参与盘后（流动性差，成本高）
- ✅ 建议: 等开盘后观察走势再决策

---

### 3. "利好出尽即利空"

**市场规律**:
```
财报前 → 预期发酵 → 股价提前上涨
财报后 → 利好落地 → 获利盘了结 → 股价回落
```

**典型案例**:
- 腾讯 2023 Q2: 财报超预期 15%，盘后涨 8%，次日开盘跌 3%
- 阿里 2024 Q1: 财报超预期 10%，盘后涨 5%，次日开盘持平

**智谱预测**:
- 盘后涨幅 12% 已反映大部分利好
- 开盘后可能回落至 +5-8% (HK$44-45)

## 实战决策

### ✅ 财报质量高
1. **双超预期** - EPS 和营收都超预期
2. **指引上调** - Q4 盈亏平衡（重大拐点）
3. **毛利率提升** - 65% → 72% (商业化质量高)
4. **客户质量** - 央企/金融机构签约增加

### ⚠️ 追高风险
1. **盘后涨幅大** - 12% 已透支利好
2. **估值偏贵** - P/S 85x，高于行业平均 50x
3. **板块轮动** - AI 热度可能降温
4. **流动性风险** - 日均成交额仅 3 亿港币

### 🎯 操作策略

**策略 A: 不追高，等回调 (推荐)**
```
开盘观察涨幅:
- 如果 < +5%: 买入 (利好未充分反映)
- 如果 +5-10%: 观望 (合理涨幅)
- 如果 > +10%: 不买 (过度反应)
```

**策略 B: 盘后小仓位试探 (激进)**
```
盘后买入 10% 仓位 @ HK$47
开盘后:
- 涨幅持续: 加仓至 30%
- 涨幅回落: 止损出局
```

**策略 C: 等待下次回调 (保守)**
```
财报利好通常在 3-5 天内消化完毕
等待回调至 HK$40-42 再买入
```
