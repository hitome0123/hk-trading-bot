## 二、选股系统（Minervini SEPA趋势模板）

### Stage 2 上升趋势筛选条件

必须同时满足以下8个条件：

| # | 条件 | 公式 |
|---|------|------|
| 1 | 现价 > MA50 | Close > SMA(50) |
| 2 | 现价 > MA150 | Close > SMA(150) |
| 3 | 现价 > MA200 | Close > SMA(200) |
| 4 | MA50 > MA150 > MA200 | 均线多头排列 |
| 5 | MA200 连续上升至少1个月 | MA200今日 > MA200_20日前 |
| 6 | 现价在52周高点25%范围内 | Close >= High_52w × 0.75 |
| 7 | 现价至少高于52周低点30% | Close >= Low_52w × 1.30 |
| 8 | 相对强度(RS) ≥ 70 | 跑赢70%以上股票 |

### VCP波动收缩形态（加分项）

- 价格波动逐次收窄（如 20% → 12% → 6%）
- 成交量逐次萎缩
- 收缩次数2-4次为佳
- 突破前量缩至地量

由于港股数据获取限制，简化版筛选：

```python
# 简化版Minervini筛选
def minervini_filter(code):
    """
    返回True表示通过Minervini筛选
    """
    kline = get_kline(code, 200)
    if len(kline) < 200:
        return False

    close = kline['close'].iloc[-1]
    ma50 = kline['close'].tail(50).mean()
    ma150 = kline['close'].tail(150).mean()
    ma200 = kline['close'].mean()

    high_52w = kline['high'].max()
    low_52w = kline['low'].min()

    # 8条件检查
    conditions = [
        close > ma50,                    # 1. 现价>MA50
        close > ma150,                   # 2. 现价>MA150
        close > ma200,                   # 3. 现价>MA200
        ma50 > ma150 > ma200,           # 4. 均线多头
        kline['close'].tail(20).mean() > kline['close'].iloc[-40:-20].mean(),  # 5. MA200上升
        close >= high_52w * 0.75,       # 6. 接近52周高
        close >= low_52w * 1.30,        # 7. 高于52周低30%
    ]

    return sum(conditions) >= 6  # 至少满足6条
```

## 三、关键位识别（Livermore关键点）

### 关键点类型

| 类型 | 定义 | 作用 |
|------|------|------|
| **初始关键点** | 趋势开始的起点 | 设定止损位 |
| **持续关键点** | 趋势中的回调低点 | 加仓/止损参考 |
| **反转关键点** | 趋势可能结束的点 | 减仓/清仓信号 |

### Livermore关键价位计算

```python
def livermore_pivot_points(kline):
    """
    计算Livermore风格的关键价位
    """
    # 近期高低点
    high_5d = kline['high'].tail(5).max()
    low_5d = kline['low'].tail(5).min()
    high_20d = kline['high'].tail(20).max()
    low_20d = kline['low'].tail(20).min()

    # 关键价位
    pivots = {
        'resistance_1': high_5d,         # 短期阻力
        'resistance_2': high_20d,        # 中期阻力
        'support_1': low_5d,             # 短期支撑
        'support_2': low_20d,            # 中期支撑
        'breakout_buy': high_20d * 1.01, # 突破买入点
        'breakdown_sell': low_20d * 0.99 # 跌破卖出点
    }

    return pivots
```

### Livermore交易法则

1. **只在关键点交易**：等待价格到达关键位再行动
2. **确认突破**：突破关键位+放量才入场
3. **坐稳赢利**：赚钱靠的是"坐着"，不是频繁交易
4. **永不摊平亏损**：亏损时只能减仓，不能加仓

## 四、入场信号（Larry Williams波动率突破）

### 核心公式

```
入场价 = 昨日收盘 + (昨日振幅 × K系数)
昨日振幅 = 昨日最高 - 昨日最低
K系数 = 0.6（默认）
```

### 多头入场条件

```python
def williams_breakout_signal(kline):
    """
    Larry Williams波动率突破信号
    """
    yesterday = kline.iloc[-2]
    today = kline.iloc[-1]

    # 计算波动率
    yesterday_range = yesterday['high'] - yesterday['low']

    # 计算突破价
    K = 0.6  # 噪音过滤系数
    breakout_price = yesterday['close'] + (yesterday_range * K)
    breakdown_price = yesterday['close'] - (yesterday_range * K)

    # 入场信号
    if today['high'] >= breakout_price and today['close'] > yesterday['close']:
        return 'BUY', breakout_price
    elif today['low'] <= breakdown_price and today['close'] < yesterday['close']:
        return 'SELL', breakdown_price
    else:
        return 'HOLD', None
```

### 止损止盈设置

| 设置 | 公式 | 说明 |
|------|------|------|
| **止损** | 入场价 - 2×突破区间 | 风险控制 |
| **止盈** | 入场价 + 2×突破区间 | 盈亏比1:1 |
| **时间止损** | 收盘前平仓 | 日内交易原则 |

### 港股T+0增强版

```python
def williams_t0_strategy(code):
    """
    港股T+0波动率突破策略
    """
    kline = get_kline(code, 5)
    yesterday = kline.iloc[-2]

    # 波动率计算
    atr = (kline['high'] - kline['low']).mean()  # 5日ATR
    yesterday_range = yesterday['high'] - yesterday['low']

    # 动态K系数（根据波动率调整）
    if yesterday_range > atr * 1.5:
        K = 0.5  # 高波动降低灵敏度
    elif yesterday_range < atr * 0.5:
        K = 0.7  # 低波动提高灵敏度
    else:
        K = 0.6  # 正常波动

    # 计算入场价
    entry_long = yesterday['close'] + (yesterday_range * K)
    entry_short = yesterday['close'] - (yesterday_range * K)

    # T+0止盈止损
    stop_loss = yesterday_range * 1.5  # 1.5倍波动率止损
    take_profit = yesterday_range * 2  # 2倍波动率止盈

    return {
        'entry_long': entry_long,
        'entry_short': entry_short,
        'stop_loss': stop_loss,
        'take_profit': take_profit,
        'K': K
    }
```

## 五、趋势反转确认（Sperandeo 123法则）

### 123反转定义

**上升趋势反转为下降：**

| 步骤 | 条件 | 说明 |
|------|------|------|
| **1** | 趋势线被跌破 | 上升趋势线被向下突破 |
| **2** | 价格回测但不创新高 | 反弹无法突破前高 |
| **3** | 跌破回调低点 | 确认反转完成 |

**下降趋势反转为上升：**

| 步骤 | 条件 | 说明 |
|------|------|------|
| **1** | 趋势线被突破 | 下降趋势线被向上突破 |
| **2** | 价格回调但不创新低 | 回调守住前低 |
| **3** | 突破回调高点 | 确认反转完成 |

### 2B法则（加强版）

当价格刚刚突破前高/前低，但立即反转，是强烈的反转信号。

```python
def sperandeo_123_reversal(kline):
    """
    Sperandeo 123反转法则检测
    """
    if len(kline) < 20:
        return None

    closes = kline['close'].values
    highs = kline['high'].values
    lows = kline['low'].values

    # 找最近的高低点
    recent_high_idx = highs[-10:].argmax() + len(highs) - 10
    recent_low_idx = lows[-10:].argmin() + len(lows) - 10

    current_price = closes[-1]
    recent_high = highs[recent_high_idx]
    recent_low = lows[recent_low_idx]

    # 判断是否处于反转过程
    # 条件1：趋势线突破（简化为跌破/突破20日均线）
    ma20 = closes[-20:].mean()

    # 上升趋势反转信号
    if recent_high_idx < recent_low_idx:  # 高点在低点之前 = 可能下跌
        if current_price < ma20:  # 跌破趋势线
            # 检查2B信号：刚创新高又跌回
            if highs[-3:].max() >= recent_high * 0.995:  # 接近或突破前高
                if current_price < recent_low:  # 跌破前低
                    return 'SELL_REVERSAL'  # 123反转卖出信号
                elif current_price < closes[-2]:  # 开始回落
                    return 'SELL_WARNING'  # 潜在反转警告

    # 下降趋势反转信号
    if recent_low_idx < recent_high_idx:  # 低点在高点之前 = 可能上涨
        if current_price > ma20:  # 突破趋势线
            if lows[-3:].min() <= recent_low * 1.005:  # 接近或跌破前低
                if current_price > recent_high:  # 突破前高
                    return 'BUY_REVERSAL'  # 123反转买入信号
                elif current_price > closes[-2]:  # 开始上涨
                    return 'BUY_WARNING'  # 潜在反转警告

    return None
```

### 综合评分系统

```python
def master_strategy_score(code):
    """
    四大师综合策略评分
    满分100分
    """
    score = 0
    signals = []

    kline = get_kline(code, 200)

    # 1. Minervini趋势模板 (25分)
    if minervini_filter(code):
        score += 25
        signals.append('✅ Minervini: Stage 2上升趋势')
    else:
        signals.append('❌ Minervini: 不在Stage 2')

    # 2. Livermore关键位 (20分)
    pivots = livermore_pivot_points(kline)
    current = kline['close'].iloc[-1]
    if current <= pivots['support_1'] * 1.02:  # 接近支撑
        score += 20
        signals.append(f'✅ Livermore: 接近支撑位 {pivots["support_1"]:.2f}')
    elif current >= pivots['resistance_1'] * 0.98:  # 接近阻力
        score += 10
        signals.append(f'⚠️ Livermore: 接近阻力位 {pivots["resistance_1"]:.2f}')

    # 3. Williams波动率突破 (25分)
    signal, entry = williams_breakout_signal(kline)
    if signal == 'BUY':
        score += 25
        signals.append(f'✅ Williams: 波动率突破买入 @ {entry:.2f}')
    elif signal == 'SELL':
        score -= 10
        signals.append('❌ Williams: 波动率突破卖出信号')

    # 4. Sperandeo反转检测 (15分)
    reversal = sperandeo_123_reversal(kline)
    if reversal is None:
        score += 15
        signals.append('✅ Sperandeo: 无反转信号')
    elif 'BUY' in reversal:
        score += 20
        signals.append('✅✅ Sperandeo: 123反转买入信号!')
    elif 'SELL' in reversal:
        score -= 20
        signals.append('❌❌ Sperandeo: 123反转卖出信号!')

    # 5. VWAP成本分析 (15分)
    vwap_20 = calculate_vwap(kline.tail(20))
    vs_vwap = (current / vwap_20 - 1) * 100
    if vs_vwap <= -5:
        score += 15
        signals.append(f'✅ VWAP: 低于成本{abs(vs_vwap):.1f}%')
    elif vs_vwap <= 0:
        score += 10
        signals.append(f'✅ VWAP: 接近成本')
    elif vs_vwap > 20:
        score -= 10
        signals.append(f'⚠️ VWAP: 高于成本{vs_vwap:.1f}%')

    return {
        'score': score,
        'signals': signals,
        'recommendation': get_recommendation(score)
    }

def get_recommendation(score):
    if score >= 70:
        return '🟢 强烈买入'
    elif score >= 50:
        return '🟢 可以买入'
    elif score >= 30:
        return '🟡 观望为主'
    else:
        return '🔴 不建议买入'
```
