# 如何添加新用户到Polymarket跟单Bot

## 方案1：多人共享推送（推荐）

一个bot推送给多个人。

### 步骤1：朋友获取chat_id

**朋友需要做：**

1. 在Telegram搜索你的bot并发送 `/start`
   - Bot用户名：在你的Telegram找到之前创建的bot

2. 获取chat_id（两种方法）：

   **方法A - 朋友自己获取（如果他有Python环境）：**
   ```bash
   cd /path/to/polymarket-bot
   python3 src/get_chat_id.py
   ```
   会输出：`Chat ID: 1234567890`

   **方法B - 你帮他获取：**
   让朋友给bot发个消息后，你运行：
   ```bash
   cd /Users/mantou/hk-trading-bot/polymarket-bot
   python3 src/get_chat_id.py
   ```
   会看到所有对话的chat_id

### 步骤2：添加到配置文件

编辑 `config.yaml`：

```yaml
notifications:
  telegram:
    enabled: true
    bot_token: "8590123130:AAGu-7p7AUDmZm90M8-svKpTSLUC-VCs80o"
    # 多个用户用逗号分隔
    chat_id: "7082819163,朋友的chat_id,另一个朋友的chat_id"
```

例如：
```yaml
chat_id: "7082819163,1234567890,9876543210"
```

### 步骤3：重启bot

```bash
# 停止旧bot
pkill -9 -f "python3 main.py"

# 启动新bot
cd /Users/mantou/hk-trading-bot/polymarket-bot
export $(cat .env | grep -v '^#' | xargs)
nohup python3 main.py > logs/bot_live.log 2>&1 &

# 确认启动
ps aux | grep "python3 main.py" | grep -v grep
```

### 步骤4：测试

bot启动后，所有用户都会收到启动通知：

## 方案2：朋友独立部署（完全隔离）

如果朋友想要自己的配置（不同的资金、不同的Sharp交易员），可以完整部署一份。

### 步骤：

1. **复制整个项目**
   ```bash
   # 打包
   cd /Users/mantou/hk-trading-bot
   tar -czf polymarket-bot.tar.gz polymarket-bot/

   # 发给朋友，朋友解压
   tar -xzf polymarket-bot.tar.gz
   cd polymarket-bot
   ```

2. **朋友创建自己的Telegram bot**
   - 在Telegram搜索 `@BotFather`
   - 发送 `/newbot` 创建新bot
   - 获取新的 `bot_token`

3. **朋友获取自己的chat_id**
   ```bash
   python3 src/get_chat_id.py
   ```

4. **朋友修改config.yaml**
   - 填入自己的 `bot_token` 和 `chat_id`
   - 调整参数（资金、Sharp交易员地址等）

5. **朋友设置私钥**
   ```bash
   echo "WALLET_PRIVATE_KEY=0x朋友的私钥" > .env
   ```

6. **朋友启动bot**
   ```bash
   pip3 install -r requirements-core.txt
   python3 main.py
   ```

## 推荐使用方案1

- ✅ 简单：只需添加chat_id
- ✅ 统一：大家收到相同的交易信号
- ✅ 维护方便：只需要管理一个bot
- ✅ 成本低：共享同一个Sharp交易员数据

## 常见问题

### Q: 如何删除某个用户？
A: 在config.yaml中删除他的chat_id，重启bot即可。

### Q: 不同用户可以有不同的资金配置吗？
A: 方案1不行，大家收到的是相同的推荐金额。如果需要个性化，用方案2独立部署。

### Q: 用户数量有限制吗？
A: Telegram bot没有限制，理论上可以推送给无限多用户。

### Q: 如何暂停某个用户的推送？
A: 临时方案：删除他的chat_id并重启。永久方案：需要修改代码支持用户黑名单。

## 安全建议

⚠️ **不要分享你的私钥（.env文件）**
⚠️ **不要分享bot_token（除非大家共用同一个bot）**
✅ **只分享chat_id是安全的**
