### 2.2 Minervini SEPA 趋势模板（Stage 2筛选）

#### 必须同时满足的8个条件：

| # | 条件 | 公式 | 作用 |
|---|------|------|------|
| 1 | 现价 > MA50 | Close > SMA(50) | 中期趋势向上 |
| 2 | 现价 > MA150 | Close > SMA(150) | 长期趋势向上 |
| 3 | 现价 > MA200 | Close > SMA(200) | 超长期趋势向上 |
| 4 | 均线多头排列 | MA50 > MA150 > MA200 | 趋势健康 |
| 5 | MA200 上升 | MA200今日 > MA200_20日前 | 长期趋势改善 |
| 6 | 接近52周高 | Close ≥ High_52w × 0.75 | 强势股 |
| 7 | 高于52周低 | Close ≥ Low_52w × 1.30 | 远离底部 |
| 8 | 相对强度 | RS ≥ 70 | 跑赢70%股票 |

#### 港股简化版（至少满足6条）：

```python
def minervini_filter(code):
    kline = get_kline(code, 200)
    close = kline['close'].iloc[-1]
    ma50 = kline['close'].tail(50).mean()
    ma150 = kline['close'].tail(150).mean()
    ma200 = kline['close'].mean()
    high_52w = kline['high'].max()
    low_52w = kline['low'].min()

    conditions = [
        close > ma50,                    # 1
        close > ma150,                   # 2
        close > ma200,                   # 3
        ma50 > ma150 > ma200,           # 4
        kline['close'].tail(20).mean() > kline['close'].iloc[-40:-20].mean(),  # 5
        close >= high_52w * 0.75,       # 6
        close >= low_52w * 1.30,        # 7
    ]

    return sum(conditions) >= 6
```

### 2.3 Livermore 关键点交易法

#### 三类关键点：

| 类型 | 定义 | 作用 |
|------|------|------|
| **初始关键点** | 趋势开始的起点 | 设定止损位 |
| **持续关键点** | 趋势中的回调低点 | 加仓/止损参考 |
| **反转关键点** | 趋势可能结束的点 | 减仓/清仓信号 |

#### Livermore 四大交易法则：

1. **只在关键点交易**：等待价格到达关键位再行动
2. **确认突破**：突破关键位+放量才入场
3. **坐稳赢利**：赚钱靠的是"坐着"，不是频繁交易
4. **永不摊平亏损**：亏损时只能减仓，不能加仓

### 2.4 Larry Williams 波动率突破

#### 核心公式：

```
入场价 = 昨日收盘 + (昨日振幅 × K系数)
昨日振幅 = 昨日最高 - 昨日最低
K系数 = 0.6（默认）

止损 = 入场价 - 2×突破区间
止盈 = 入场价 + 2×突破区间
```

#### 动态K系数（港股T+0增强版）：

```python
yesterday_range = yesterday['high'] - yesterday['low']
atr = (kline['high'] - kline['low']).mean()  # 5日ATR

if yesterday_range > atr * 1.5:
    K = 0.5  # 高波动降低灵敏度
elif yesterday_range < atr * 0.5:
    K = 0.7  # 低波动提高灵敏度
else:
    K = 0.6  # 正常波动

entry_long = yesterday['close'] + (yesterday_range * K)
```

### 2.5 Sperandeo 123反转法则

#### 上升趋势反转为下降：

| 步骤 | 条件 | 说明 |
|------|------|------|
| **1** | 趋势线被跌破 | 上升趋势线被向下突破 |
| **2** | 价格回测但不创新高 | 反弹无法突破前高 |
| **3** | 跌破回调低点 | 确认反转完成 → **立即卖出** |

#### 2B法则（加强版）：

当价格刚刚突破前高/前低，但立即反转，是强烈的反转信号。

```
出现2个信号 → 减仓50%
出现3个信号 → 全部离场
```

### 4.1 综合入场评分系统（满分100分）

```python
def master_entry_score(code):
    score = 0
    signals = []

    # 1. Minervini趋势 (25分)
    if minervini_stage2(code):
        score += 25
        signals.append('✅ Minervini: Stage 2上升趋势')
    else:
        signals.append('❌ Minervini: 不在Stage 2')

    # 2. Livermore关键位 (20分)
    if near_support(code):
        score += 20
        signals.append('✅ Livermore: 接近支撑位')
    elif breakout_resistance(code):
        score += 15
        signals.append('✅ Livermore: 突破阻力位')

    # 3. Williams突破 (25分)
    if williams_breakout_signal(code) == 'BUY':
        score += 25
        signals.append('✅ Williams: 波动率突破买入')

    # 4. Sperandeo无反转 (15分)
    reversal = sperandeo_123_reversal(code)
    if reversal is None:
        score += 15
        signals.append('✅ Sperandeo: 无反转信号')
    elif 'BUY' in reversal:
        score += 20
        signals.append('✅✅ Sperandeo: 123反转买入!')
    elif 'SELL' in reversal:
        score -= 20
        signals.append('❌❌ Sperandeo: 反转卖出警告!')

    # 5. VWAP成本 (15分)
    vs_vwap = current_vs_vwap(code)
    if vs_vwap <= -5:
        score += 15
        signals.append(f'✅ VWAP: 低于成本{abs(vs_vwap):.1f}%')
    elif vs_vwap <= 0:
        score += 10
        signals.append('✅ VWAP: 接近成本')
    elif vs_vwap > 20:
        score -= 10
        signals.append(f'⚠️ VWAP: 高于成本{vs_vwap:.1f}%')

    return {
        'score': score,
        'signals': signals,
        'recommendation': get_recommendation(score)
    }

def get_recommendation(score):
    if score >= 70:
        return '🟢 强烈买入'
    elif score >= 50:
        return '🟢 可以买入'
    elif score >= 30:
        return '🟡 观望为主'
    else:
        return '🔴 不建议买入'
```

### 4.3 VWAP 主力成本分析

#### 计算公式：

```python
# 20日VWAP（最常用）
kline['typical_price'] = (kline['high'] + kline['low'] + kline['close']) / 3
kline['amount'] = kline['typical_price'] * kline['volume']
vwap_20 = kline['amount'].sum() / kline['volume'].sum()

vs_vwap = (current_price / vwap_20 - 1) * 100
```

#### 解读标准：

| 现价 vs VWAP | 解读 | 建议 |
|--------------|------|------|
| < -10% | 远低于成本 | ✅✅ 超跌机会，主力被套有护盘动力 |
| -10% ~ 0% | 低于成本 | ✅ 买入有安全边际 |
| 0% ~ 15% | 接近成本 | 🟡 正常区间 |
| 15% ~ 30% | 高于成本 | ⚠️ 追高有风险，主力盈利可能出货 |
| > 30% | 远高于成本 | ❌ 不建议追 |

### 9.2 计算VWAP成本

```python
def calc_vwap(code, days=20):
    quote_ctx = OpenQuoteContext(host='127.0.0.1', port=11111)
    ret, kline = quote_ctx.request_history_kline(code, max_count=days)
    quote_ctx.close()

    if ret == RET_OK:
        kline['typical'] = (kline['high'] + kline['low'] + kline['close']) / 3
        kline['amount'] = kline['typical'] * kline['volume']
        vwap = kline['amount'].sum() / kline['volume'].sum()
        return vwap
    return None

# 使用
vwap = calc_vwap('HK.09880', 20)
current_price = 138.80
vs_vwap = (current_price / vwap - 1) * 100
print(f"20日VWAP: {vwap:.2f}, vs VWAP: {vs_vwap:+.1f}%")
```

### 9.3 Williams 突破价计算

```python
def williams_breakout_price(code):
    quote_ctx = OpenQuoteContext(host='127.0.0.1', port=11111)
    ret, kline = quote_ctx.request_history_kline(code, max_count=5)
    quote_ctx.close()

    yesterday = kline.iloc[-2]
    yesterday_range = yesterday['high'] - yesterday['low']

    # 动态K系数
    atr = (kline['high'] - kline['low']).mean()
    if yesterday_range > atr * 1.5:
        K = 0.5
    elif yesterday_range < atr * 0.5:
        K = 0.7
    else:
        K = 0.6

    breakout_price = yesterday['close'] + (yesterday_range * K)
    breakdown_price = yesterday['close'] - (yesterday_range * K)

    return {
        'entry_long': breakout_price,
        'entry_short': breakdown_price,
        'K': K,
        'yesterday_close': yesterday['close'],
        'yesterday_range': yesterday_range
    }

# 使用
signal = williams_breakout_price('HK.09880')
print(f"Williams突破买入价: {signal['entry_long']:.2f}")
print(f"昨收: {signal['yesterday_close']:.2f}, K系数: {signal['K']:.2f}")
```

## 十三、做T方法论（日内高抛低吸）

> **核心认知**：做T不是赌博，是利用日内振幅赚取差价。关键是选对股、算准位、守纪律。

### 13.1 做T适用条件

#### 什么股票适合做T？

| 条件 | 标准 | 原因 |
|------|------|------|
| **日均振幅** | >5% | 振幅小做T没利润空间 |
| **流动性** | 日成交额>5000万 | 成交量太小挂单难成交 |
| **你有底仓** | 持仓>100股 | 港股T+0需要有底仓才能先卖后买 |
| **非单边市** | 有涨有跌 | 单边涨/跌的股不适合做T |

#### 什么情况不做T？

```
┌─────────────────────────────────────────────────────────┐
│               ❌ 不做T的情况                             │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  1. 大盘跌超1% → 全场杀跌，别接飞刀                     │
│                                                         │
│  2. 个股单边涨停 → 别做T，拿着等涨                      │
│                                                         │
│  3. 没时间盯盘 → 做T必须看盘，否则错过点位              │
│                                                         │
│  4. 振幅<3% → 做T没利润，扣完手续费白忙                 │
│                                                         │
│  5. 有重大消息 → 先观望，消息落地再操作                 │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 13.2 关键价位计算法

#### 方法1：日内高低点法（最简单）

```
阻力位 = 今日/昨日最高价
支撑位 = 今日/昨日最低价

做T区间 = 支撑位 ~ 阻力位
```

#### 方法2：5日均价法

```
5日均价 = (近5日收盘价之和) ÷ 5

现价 > 5日均价 → 高位，优先考虑卖
现价 < 5日均价 → 低位，优先考虑买
```

#### 方法3：振幅预测法

```
预测今日振幅 = 近5日平均振幅

预测低点 = 开盘价 × (1 - 预测振幅/2)
预测高点 = 开盘价 × (1 + 预测振幅/2)
```

#### 方法4：VWAP成本线法

```
VWAP = 成交额 ÷ 成交量 (富途可直接看)

现价 > VWAP → 短线高位，可卖
现价 < VWAP → 短线低位，可买
```

### 13.3 做T两种模式

#### 模式A：正T（先买后卖）

```
适用：股价在支撑位附近

操作流程：
1. 低位买入100股（加仓）
2. 等股价上涨
3. 高位卖出100股（原底仓）
4. 持仓数量不变，成本降低
```

#### 模式B：倒T（先卖后买）

```
适用：股价在阻力位附近

操作流程：
1. 高位卖出100股（减底仓）
2. 等股价下跌
3. 低位买回100股
4. 持仓数量不变，成本降低
```

#### 选择哪种模式？

```
开盘位置判断：

开盘价 > 昨收 + 2% → 高开，用倒T（先卖）
开盘价 < 昨收 - 2% → 低开，用正T（先买）
开盘价 ≈ 昨收     → 等30分钟看方向
```

### 13.4 做T仓位管理

#### 铁律：只用部分仓位做T

```
总持仓: 200股

做T仓位: 100股（50%）
底仓保留: 100股（50%）

原因：
- 万一做T失败，还有底仓
- 避免被完全洗出去
```

#### 仓位比例建议

| 振幅 | 做T仓位 | 底仓 | 风险 |
|:----:|:-------:|:----:|:----:|
| >10% | 30-50% | 50-70% | 高振幅=高风险 |
| 5-10% | 50% | 50% | 正常 |
| 3-5% | 30% | 70% | 低振幅少做 |

### 13.5 做T执行流程

```
盘前准备（08:00-09:30）
│
├─ 1. 看昨日K线，标出高低点
├─ 2. 计算5日均价
├─ 3. 计算预测振幅
├─ 4. 确定今日支撑/阻力位
└─ 5. 决定今日做T目标股

开盘观察（09:30-10:00）
│
├─ 1. 观察开盘价位置（高开/低开/平开）
├─ 2. 观察前30分钟走势
├─ 3. 确定做正T还是倒T
└─ 4. 挂单，不追涨杀跌

执行做T（10:00-15:30）
│
├─ 正T：低位挂买单 → 成交 → 等反弹 → 高位挂卖单
├─ 倒T：高位挂卖单 → 成交 → 等回调 → 低位挂买单
└─ 未成交就撤单，不勉强

收盘复盘（15:30-16:00）
│
├─ 1. 记录今日做T结果
├─ 2. 对比实际振幅vs预测
├─ 3. 分析哪里做对/做错
└─ 4. 调整明日策略
```

### 13.6 做T止损规则

#### 正T止损（先买后卖）

```
买入后股价继续跌：

止损位 = 买入价 × (1 - 3%)

例如：46元买入，跌破44.62就止损
```

#### 倒T止损（先卖后买）

```
卖出后股价继续涨：

止损位 = 卖出价 × (1 + 5%)

例如：50元卖出，涨过52.5就认亏买回
```

#### 做T失败处理

```
做T亏了怎么办？

1. 立即止损，不死扛
2. 当天不再做T这只股
3. 复盘分析为什么失败
4. 调整明日策略
```

### 13.7 做T收益计算

#### 单次做T收益

```
收益 = (卖出价 - 买入价) × 做T股数 - 手续费

手续费 ≈ 成交金额 × 0.3%（港股印花税+佣金）
```

#### 做T盈亏平衡点

```
最小做T差价 = 股价 × 0.6%

例如：股价100元
最小差价 = 0.6元
低于0.6元的差价做T不赚钱
```

#### 每日做T目标

```
假设：
- 持仓市值 10,000元
- 做T仓位 50% = 5,000元
- 日均振幅 6%

理论收益 = 5,000 × 6% ÷ 2 = 150元/天
实际收益 ≈ 100元/天（打个折）
```

### 13.8 做T实战案例（方法论角度）

#### 案例：如何做一只振幅10%的股

```
假设条件：
- 持仓200股，成本50元
- 昨日振幅10%（低48，高53）
- 今日预期振幅10%

步骤1：计算关键价位
- 支撑位 = 48（昨日低点）
- 阻力位 = 53（昨日高点）
- 5日均价 = 50.5

步骤2：开盘判断
- 开盘价51 > 昨收50 → 小幅高开
- 现价 > 均价 → 偏高位

步骤3：选择模式
- 高开偏高位 → 用倒T（先卖后买）

步骤4：执行
- 09:35 挂卖单52.5（阻力位下方）
- 10:30 成交，卖出100股@52.5
- 等待回调...
- 14:00 挂买单49（支撑位上方）
- 14:45 成交，买入100股@49

步骤5：计算收益
- 卖出 52.5 × 100 = 5,250
- 买入 49 × 100 = 4,900
- 差价收益 = 350元
- 手续费 ≈ 30元
- 净收益 = 320元 ✅
```
