## 为什么这样设计？

| 组件 | 部署位置 | 原因 |
|------|---------|------|
| 交易监控脚本 | 云服务器 | 7x24运行，不怕断电/断网 |
| 富途OpenD | 本地Mac | **必须本地，需要GUI + 账号登录** |

**富途OpenD无法部署到云服务器**，因为：
1. 需要GUI界面登录
2. 需要手机扫码认证
3. 需要本地行情权限

所以最优方案：**云服务器跑逻辑 + 本地Mac跑富途 + 内网穿透连接**

### Step 5: 配置内网穿透（frp）

**为什么要内网穿透？**
云服务器需要访问你本地Mac的富途OpenD（127.0.0.1:11111）

#### 5.1 在阿里云服务器安装frps（服务端）

```bash
# 下载frp
cd /opt
wget https://github.com/fatedier/frp/releases/download/v0.51.0/frp_0.51.0_linux_amd64.tar.gz
tar -xzf frp_0.51.0_linux_amd64.tar.gz
mv frp_0.51.0_linux_amd64 frp
cd frp

# 创建服务端配置
cat > frps.ini <<'EOF'
[common]
bind_port = 7000
vhost_http_port = 8080
EOF

# 启动服务端
nohup ./frps -c frps.ini > frps.log 2>&1 &

# 检查是否启动
netstat -tuln | grep 7000
# 应该看到：tcp 0 0.0.0.0:7000 LISTEN
```

#### 5.2 在本地Mac安装frpc（客户端）

```bash
# 下载frp
cd ~/Downloads
wget https://github.com/fatedier/frp/releases/download/v0.51.0/frp_0.51.0_darwin_amd64.tar.gz
tar -xzf frp_0.51.0_darwin_amd64.tar.gz
cd frp_0.51.0_darwin_amd64

# 创建客户端配置
cat > frpc.ini <<'EOF'
[common]
server_addr = 你的服务器IP
server_port = 7000

[futu]
type = tcp
local_ip = 127.0.0.1
local_port = 11111
remote_port = 11111
EOF

# 启动客户端
nohup ./frpc -c frpc.ini > frpc.log 2>&1 &
```

### Step 6: 创建监控脚本（服务器）

```bash
cd /opt/trading-bot
cat > monitor_service.py <<'EOF'
#!/usr/bin/env python3
"""
交易监控服务 - 运行在阿里云服务器
"""
import sys
import time
import subprocess
from datetime import datetime

# 富途OpenD地址（通过内网穿透）
FUTU_HOST = "127.0.0.1"  # frp会转发到本地Mac
FUTU_PORT = 11111

def is_trading_hours():
    """判断是否交易时间"""
    now = datetime.now()
    hour = now.hour
    weekday = now.weekday()

    if weekday >= 5:  # 周末
        return False

    # 港股交易时间：9:30-16:00
    if 9 <= hour <= 15:
        return True

    return False

def run_coal_analysis():
    """运行煤炭分析"""
    try:
        print(f"[{datetime.now()}] 🔍 运行煤炭ETF分析...")
        subprocess.run([
            'python3',
            'coal_etf_analysis.py'
        ], timeout=120)
        print(f"[{datetime.now()}] ✅ 煤炭分析完成")
    except Exception as e:
        print(f"[{datetime.now()}] ❌ 错误: {e}")

def main():
    print(f"[{datetime.now()}] 🚀 交易监控服务启动（阿里云服务器）")
    print("=" * 60)

    while True:
        if is_trading_hours():
            run_coal_analysis()
            time.sleep(300)  # 5分钟扫描一次
        else:
            print(f"[{datetime.now()}] 💤 非交易时间，等待中...")
            time.sleep(3600)  # 1小时检查一次

if __name__ == '__main__':
    main()
EOF

chmod +x monitor_service.py
```

### Step 7: 配置systemd守护进程

```bash
# 创建systemd服务
cat > /etc/systemd/system/trading-monitor.service <<'EOF'
[Unit]
Description=Trading Monitor Service
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/opt/trading-bot
ExecStart=/opt/trading-bot/venv/bin/python3 /opt/trading-bot/monitor_service.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
EOF

# 启动服务
systemctl daemon-reload
systemctl enable trading-monitor.service
systemctl start trading-monitor.service

# 查看状态
systemctl status trading-monitor.service
```

## 六、常见问题

### Q1: 服务器连不上富途OpenD

**检查步骤：**
```bash
# 1. 检查frp服务端是否运行
ps aux | grep frps

# 2. 检查frp客户端是否运行（Mac上）
ps aux | grep frpc

# 3. 检查端口
netstat -tuln | grep 11111

# 4. 重启frp
systemctl restart frp
```

### Q2: 分析脚本报错

**查看日志：**
```bash
journalctl -u trading-monitor -n 50
```

### Q3: 本地Mac关机了怎么办？

**富途必须本地运行，Mac关机=无法获取行情**

**解决方案：**
- Mac设置不休眠
- 或买Mac mini专门跑富途
