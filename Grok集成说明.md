## 💡 工作原理

### Grok的优势
1. **实时X/Twitter数据** - 直接访问全球最快的讨论平台
2. **情绪分析准确** - AI理解微妙的情绪变化
3. **热度量化** - 0-100分客观评分

### 分析维度
| 维度 | 说明 | 阈值 |
|------|------|------|
| 板块热度 | 整体讨论热度 | >50才显示 |
| 个股热度 | 单个股票热度 | >30才显示 |
| 社区情绪 | positive/neutral/negative | - |
| 热议话题 | 最热3个话题 | - |

### 触发逻辑
```python
if os.getenv('GROK_API_KEY'):
    # 分析板块整体
    grok.analyze_sector_sentiment(sector_name, top_stocks)

    # 筛选高热度信息
    if heat_score > 50:  # 板块
        添加到报告
    if heat_score > 30:  # 个股
        添加到报告
```

## ⚠️ 注意事项

### 1. API配额
- Grok API是**付费服务**
- 根据使用量计费
- 建议监控使用情况

### 2. 响应时间
- Grok需要搜索X/Twitter
- 每次分析约**7-15秒**
- 会增加整体运行时间

### 3. 可选功能
如果API Key未配置：
- 系统自动跳过Grok分析
- 不影响其他数据源
- **优雅降级**

### 4. 错误处理
```python
try:
    grok = GrokSentimentAnalyzer()
    results = grok.analyze_sector_sentiment(...)
except Exception as e:
    print(f"⚠️ Grok分析失败: {e}")
    # 继续运行，不中断系统
```
