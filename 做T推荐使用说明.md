# 港股做T推荐系统使用说明

## 🎯 核心策略逻辑

### 分析流程
1. **扫描热门板块**（涨幅榜TOP10）
2. **分析18只优质大盘股**（市值大、基本面好、流动性强）
3. **完整技术分析**
   - RSI指标（超买超卖）
   - MACD指标（金叉死叉）
   - 量比（放量缩量）
   - 社交热度（雪球、东财股吧）
4. **智能评分**（总分100分）
   - **放量上涨优先**（最高40分）- 量比≥2且涨幅>0
   - 振幅评分（20分）- 振幅越大越适合做T
   - RSI评分（15分）- RSI<30最佳
   - MACD评分（15分）- 金叉加分
   - 社交热度（10分）

### 评分标准
- **≥60分**：强烈推荐 ✅
- **50-60分**：谨慎参与 ⚠️
- **<50分**：不建议 ❌

### 优先级排序
1. **放量上涨 + MACD金叉**（追涨信号）
2. **RSI超卖（<30）+ MACD金叉**（抄底信号）
3. **缩量回调 + RSI买入区**（低吸机会）
4. 其他情况

## ⚠️ 做T注意事项

1. **优先选择放量上涨的股票**（量比≥2且涨幅>0）
2. **评分≥60分**为强推荐，50-60分谨慎参与
3. **RSI<30超卖 + MACD金叉**是最佳买点
4. **放量上涨 + MACD金叉**是追涨信号
5. **缩量回调**是低吸机会
6. **严格止损**，跌破止损位立即出局
7. **单只仓位≤30%**，避开开盘/收盘各30分钟
8. 优先选择热门板块，跟随市场热点
