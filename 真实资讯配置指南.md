# 真实资讯配置指南

## 概述
盘前扫描器现已升级为**真实社区资讯版**，可以从股吧、雪球等平台抓取真实讨论内容。

## 数据源说明

### 1. 股吧（东方财富）✅ 默认开启
- **状态**: 无需配置，开箱即用
- **内容**: 散户讨论、股吧热帖
- **覆盖**: A股、港股
- **优点**: 无需登录，直接抓取

### 2. 雪球 ⚙️ 需配置Cookie
- **状态**: 需要登录配置
- **内容**: 专业投资者讨论、研报分享
- **覆盖**: A股、港股、美股
- **优点**: 内容质量高，机构观点多

### 3. 淘股吧 ⏸️ 暂未启用
- **状态**: API较复杂，暂时禁用
- **内容**: 游资观点、短线技巧
- **说明**: 未来版本可能支持

### 4. 富途公告 ✅ 默认开启
- **状态**: 通过富途OpenD API获取
- **内容**: 公司公告、财报
- **覆盖**: 港股、美股

## 配置步骤

### 方法1: 不配置（基础版）
直接使用，只能抓取股吧和部分公开内容。

```bash
# 测试基础功能
cd ~/hk-trading-bot
python3 test_real_news.py
```

### 方法2: 配置雪球Cookie（推荐）

#### 第一步：获取雪球Cookie

1. **打开雪球网站**
   ```
   https://xueqiu.com
   ```

2. **登录你的雪球账号**
   - 如果没有账号，先注册一个（免费）

3. **打开浏览器开发者工具**
   - Mac: `Command + Option + I`
   - Windows: `F12`

4. **切换到 Network（网络）标签**

5. **刷新页面**
   - 按 `Command + R` (Mac) 或 `F5` (Windows)

6. **找到任意请求**
   - 在请求列表中点击任意一个请求
   - 例如：`quote.json` 或 `search.json`

7. **复制Cookie**
   - 在右侧 Headers（请求头）部分
   - 找到 `Cookie:` 那一行
   - 复制整个cookie值（很长的一串）

   **示例cookie格式**:
   ```
   xq_a_token=abc123...; xq_r_token=def456...; xq_id_token=xyz789...; u=1234567890
   ```

#### 第二步：配置到系统

1. **编辑配置文件**
   ```bash
   cd ~/hk-trading-bot
   nano news_config.json
   ```

2. **找到 xueqiu 部分**
   ```json
   "xueqiu": {
     "enabled": false,   // 改为 true
     "cookie": "在这里填入雪球cookie"  // 粘贴你复制的cookie
   }
   ```

3. **修改后的样子**
   ```json
   "xueqiu": {
     "enabled": true,
     "cookie": "xq_a_token=abc123...; xq_r_token=def456...; xq_id_token=xyz789..."
   }
   ```

4. **保存并退出**
   - 按 `Control + X`
   - 按 `Y` 确认
   - 按 `Enter` 保存

#### 第三步：测试配置

```bash
cd ~/hk-trading-bot
python3 test_real_news.py
```


## 常见问题

### Q1: Cookie多久过期？
**A**: 通常7-30天。如果扫描器突然抓不到雪球内容了，重新获取cookie即可。

### Q2: Cookie安全吗？
**A**: Cookie只保存在你的本地文件中，不会上传到任何服务器。但建议：
- 不要分享 `news_config.json` 文件
- 定期更换cookie

### Q3: 抓取会被封号吗？
**A**: 不会。抓取器：
- 每只股票间隔0.3秒
- 每次只抓取3-5条内容
- 模拟正常浏览行为

### Q4: 股吧没内容怎么办？
**A**: 股吧可能会限制访问频率，解决方法：
1. 等待5-10分钟后重试
2. 股吧没内容不影响雪球抓取
3. 可以只依赖雪球（内容质量更高）

### Q5: 可以添加其他数据源吗？
**A**: 可以！欢迎提建议：
- 金十数据
- 同花顺
- 东方财富资讯
- 财联社
