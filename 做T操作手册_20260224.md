## 做T纪律 (必须遵守！)

### 铁律1: 分批操作
```
不要一把梭！每次只做1/3到1/2的仓位
```

### 铁律2: 设好价格等成交
```
挂单等待，不要追涨杀跌
低吸单: 挂在支撑位上方
高抛单: 挂在阻力位下方
```

### 铁律3: 日内必须平仓
```
做T = 当天买的当天卖 (或当天卖的当天买回)
不要把T仓留过夜
```

### 铁律4: 大盘不好不做T
```
大盘跌超过1% → 今天不做T
全场都在跌 → 你的股也会跌，别接飞刀
```

### 铁律5: 止损优先
```
跌破支撑2 → 立即止损，不要幻想反弹
做T亏了 → 认亏出局，不要死扛
```
