## 二、完整分析流程

### 第1步：实时行情

获取基础数据：
- 现价、涨跌幅、成交量
- 量比 = 今日成交量 / 20日均量
- 量比>1.5 = 放量，有资金关注

### 第2步：VWAP成本分析（核心指标）

**VWAP = 成交量加权平均价，代表市场平均持仓成本**

```python
# 计算20日VWAP
vwap = (收盘价 × 成交量).sum() / 成交量.sum()
vs_vwap = (现价 / vwap - 1) × 100%
```

| 现价 vs VWAP | 解读 | 建议 |
|--------------|------|------|
| < -10% | 远低于成本 | ✅✅ 超跌机会 |
| -10% ~ 0% | 低于成本 | ✅ 买入有安全边际 |
| 0% ~ 15% | 接近成本 | 🟡 正常区间 |
| 15% ~ 30% | 高于成本 | ⚠️ 追高有风险 |
| > 30% | 远高于成本 | ❌ 不建议追 |

### 第3步：技术指标

| 指标 | 计算周期 | 买入信号 | 卖出信号 |
|------|----------|----------|----------|
| **RSI** | 14日 | <30 超卖 | >70 超买 |
| **MACD** | 12/26/9 | 金叉（DIF上穿DEA） | 死叉（DIF下穿DEA） |
| **KDJ** | 9日 | K<20 超卖区 | K>80 超买区 |
| **布林带** | 20日 | 跌破下轨 | 突破上轨 |

### 第4步：均线分析

```
多头排列：MA5 > MA10 > MA20 → 上升趋势 ✅
空头排列：MA5 < MA10 < MA20 → 下降趋势 ❌
交织状态：均线纠缠 → 方向不明 🟡
```

**关键判断：**
- 站上MA5 → 短期强势
- 站上MA20 → 中期趋势健康
- 站上MA60 → 长期趋势向好

### 第5步：关键价位识别

| 类型 | 计算方法 | 用途 |
|------|----------|------|
| 支撑位1 | 5日最低价 | 短期支撑 |
| 支撑位2 | 20日最低价 | 中期支撑 |
| 支撑位3 | MA20 | 趋势支撑 |
| 阻力位1 | 5日最高价 | 短期阻力 |
| 阻力位2 | 20日最高价 | 中期阻力 |

### 第6步：综合评分（0-10分）

| 评分项 | 加分条件 | 分值 |
|--------|----------|------|
| 均线 | 站上MA5 | +1 |
| 均线 | 站上MA20 | +1 |
| 均线 | 多头排列 | +1 |
| VWAP | 低于成本 | +1~2 |
| RSI | 超卖区 | +1~2 |
| MACD | 多头/金叉 | +1 |
| 布林带 | 跌破下轨 | +1 |

**评分解读：**
- ≥7分：可以买入
- 4-6分：观望或小仓位
- <4分：不建议买入

## 三、T+0日内交易判断

### 适合做T的条件

必须同时满足：
- [x] 日波动率 ≥ 5%
- [x] 量比 ≥ 1.5（放量）
- [x] 有明确趋势（不是横盘震荡）
- [x] 流动性好（日成交额>5000万港元）

### T+0时间表

| 时间 | 阶段 | 操作 |
|------|------|------|
| 09:30-10:00 | 观察 | 看方向，不操作 |
| **10:00-11:00** | 第一波 | Williams突破入场 |
| 11:00-12:00 | 午盘 | 持有或减仓 |
| 13:00-14:00 | 观察 | 看下午趋势 |
| **14:00-15:30** | 第二波 | 顺势加仓或新开仓 |
| 15:30-16:00 | 收盘 | 全部平仓 |

### T+0资金管理

```
单笔止损：1%（250元/每25000仓位）
单日止损：2%（500元）→ 达到就停止

单笔目标：2%（500元）
单日目标：4%（1000元）

盈利后移动止盈：
  盈利1% → 止损移至成本
  盈利2% → 止损移至+1%
  盈利3% → 止损移至+2%
```

## 四、回调概率判断

当股票大涨后，判断是否会回调：

| 信号 | 权重 | 解读 |
|------|------|------|
| 现价在日内高点附近(>90%) | +2 | 冲高回落概率大 |
| 尾盘量能萎缩(<1倍) | +2 | 动能不足 |
| RSI超买(>70) | +1 | 短期过热 |
| 涨幅过大(>5%) | +1 | 获利盘回吐压力 |

**回调概率评分：**
- ≥6分：大概率回调，可先卖出
- 4-5分：可能回调，减仓观望
- <4分：继续持有

## 六、入场评分公式

```python
def entry_score(stock):
    score = 0

    # Minervini趋势 (20分)
    if stage2_uptrend(stock):
        score += 20

    # Livermore关键位 (15分)
    if near_support(stock):
        score += 15
    elif breakout_resistance(stock):
        score += 10

    # Williams突破 (20分)
    if williams_signal(stock) == 'BUY':
        score += 20

    # Sperandeo无反转 (15分)
    if no_reversal_signal(stock):
        score += 15

    # VWAP成本 (15分)
    vs_vwap = current_vs_vwap(stock)
    if vs_vwap <= -5:
        score += 15
    elif vs_vwap <= 0:
        score += 10
    elif vs_vwap > 20:
        score -= 10

    # 动量确认 (15分)
    if strong_momentum(stock):
        score += 15

    return score  # 满分100，≥70分入场
```
