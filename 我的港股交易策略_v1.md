### 铁律详解

#### 铁律1: 只做热门板块龙头

**定义**：
- 热门板块 = 当日资金净流入最多的1-2个板块
- 龙头 = 板块内涨幅第1-2名 + 成交量最大

**每日操作**：
```
08:30-09:00 查找今日热点：
1. 查看隔夜消息 → 可能的热点板块
2. hk-trade-bot 扫描 → 板块涨幅排名
3. 确定1-2个板块 → 只关注这些板块的龙头
```

**常见热门板块**：
| 板块 | 代表龙头 | 催化剂类型 |
|------|---------|-----------|
| 机器人/AI | 优必选(09880)、商汤(00020) | 政策、技术突破 |
| 新能源车 | 比亚迪(01211)、理想(02015) | 销量、政策 |
| 互联网 | 腾讯(00700)、阿里(09988) | 业绩、回购 |
| 医药 | 百济神州(06160)、信达(01801) | 新药、业绩 |

#### 铁律2: 突破前日高点 + 放量 → 才能买

**买入条件（全部满足）**：
```
□ 条件A: 股价 > 前日最高价
□ 条件B: 当前成交量 > 前日同时段 × 1.5
□ 条件C: 时间 > 10:00（观察开盘30分钟后）
□ 条件D: 主力资金净流入（hk-trade-bot确认）
```

**图解**：
```
价格
  │
  │         ┌── 突破点（买入）
  │         │
前日高 ─────┼─────────
  │       ╱│
  │      ╱ │← 放量确认
  │     ╱  │
前日低 ────────────────
  │
  └─────────────────────→ 时间
        10:00
```

**hk-trade-bot 命令**：
```python
from futu import *

quote_ctx = OpenQuoteContext(host='127.0.0.1', port=11111)

# 检查突破条件
code = 'HK.09880'
ret, snapshot = quote_ctx.get_market_snapshot([code])
ret, kline = quote_ctx.request_history_kline(code, max_count=2)

current_price = snapshot['last_price'].values[0]
prev_high = kline['high'].values[-2]  # 前日最高
volume_ratio = snapshot['volume_ratio'].values[0]

if current_price > prev_high and volume_ratio > 1.5:
    print(f"✅ {code} 突破信号！可以考虑买入")
else:
    print(f"❌ {code} 未满足条件，继续等待")

quote_ctx.close()
```

示例：
买入价: 82.50 港元
止损价: 82.50 × 0.98 = 80.85 港元
止损金额: 2.5万 × 2% = 500元（Lv.1仓位）

**执行要求**：
- ✅ 买入后**30秒内**必须设好止损单
- ✅ 止损触发**绝不撤单**
- ❌ 禁止"再等等看"
- ❌ 禁止"摊低成本"
- ❌ 禁止"它会反弹的"

## 十一、常见问题

### Q1: 开盘就涨停怎么办？
A: 不追。涨停后风险大，等回调或放弃这只。

### Q2: 买入后一直横盘不动怎么办？
A: 如果没触发止损，可以持有到15:00；如果有更好机会，可以换股。

### Q3: 止损后股价又涨回去了怎么办？
A: 这很正常。止损是保护机制，不要后悔。可以重新评估是否再次买入。

### Q4: 今天没有机会怎么办？
A: 空仓也是操作！不交易 = 不亏钱。保持耐心，明天再来。

### Q5: 连续亏损怎么办？
A: 1) 降低仓位 2) 检查是否遵守规则 3) 必要时暂停几天

### 计算VWAP
```python
from futu import *

def calc_vwap(code, days=20):
    quote_ctx = OpenQuoteContext(host='127.0.0.1', port=11111)
    ret, kline = quote_ctx.request_history_kline(code, max_count=days)
    quote_ctx.close()

    if ret == RET_OK:
        kline['typical'] = (kline['high'] + kline['low'] + kline['close']) / 3
        kline['amount'] = kline['typical'] * kline['volume']
        vwap = kline['amount'].sum() / kline['volume'].sum()
        return vwap
    return None

# 使用
vwap = calc_vwap('HK.09880', 20)
print(f"20日VWAP: {vwap:.2f}")
```

## 附录：「龙头催化 + 补涨跟随」策略

> 2026-02-12 从 MiniMax 交易中总结

### 策略逻辑

```
龙头出利好 → 板块情绪高涨 → 同板块掉队票补涨
```

### 成功案例

| 日期 | 龙头 | 龙头涨幅 | 掉队票 | 买入价 | 卖出价 | 盈利 |
|-----|------|---------|-------|-------|-------|------|
| 2026-02-11/12 | 智谱 | +28% | MiniMax | 528 | 612.5 | +16% |

### 操作步骤

1. **监控龙头消息**
   - 信息源：雪球、财联社、富途牛牛
   - 关注：AI、新能源、半导体等热门板块龙头

2. **确认板块联动**
   - 龙头涨 > 3% 才考虑
   - 检查同板块其他票走势

3. **找掉队的票**
   - 同板块但涨幅小于龙头 5%+
   - 或者逆势下跌的
   - 用命令：`python my_strategy_helper.py laggards`

4. **买入时机**
   - 掉队票企稳或小幅反弹时
   - 不追高，低位接

5. **卖出时机**
   - 补涨到接近龙头涨幅
   - 或者涨 10-20% 分批止盈
   - 龙头回调时立刻跑

### 风险提示

- 不是所有掉队都会补涨（可能是基本面有问题）
- 大盘股弹性小于小盘股
- 需要快速反应，适合盯盘的人
