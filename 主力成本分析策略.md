## 二、主力成本估算方法（VWAP）

### 计算公式

```
VWAP = Σ(典型价格 × 成交量) / Σ(成交量)
典型价格 = (最高价 + 最低价 + 收盘价) / 3
```

### 关键指标

| 指标 | 用途 |
|------|------|
| 60日VWAP | 中长线主力成本 |
| 20日VWAP | 短期主力成本（最常用） |
| 10日VWAP | 近期买入成本 |
| 放量日均价 | 主力集中建仓成本 |

### 分析逻辑

- 现价 < 20日VWAP → 主力可能被套，有护盘动力
- 现价 > 20日VWAP × 110% → 主力盈利，可能出货
- 你的成本 < 主力成本 → 优势持仓
- 你的成本 > 主力成本 → 风险较高

| 因素 | 分值 | 条件 |

| 主力净流入 | +30 | 净流入 > 1000万 |

| 涨幅不大 | +10 | 当日涨幅 < 3% |

| 接近成本价 | +10 | 现价 ≤ 20日VWAP |
| 买入强度 | +5 | 买入/卖出 > 1.2 |

### 评分标准

- ≥70分：强烈推荐
- 50-69分：值得关注
- <50分：观望

## 四、操作原则

### 买入条件

1. 主力净流入 > 500万
2. 现价接近或低于20日VWAP
3. 散户在卖、主力在买（筹码交换）
4. 当日涨幅不超过5%（留有空间）

### 卖出条件

1. 现价 > 20日VWAP × 115%（主力盈利过多）
2. 主力连续3天净流出
3. 你的盈利 > 15% 可考虑止盈
4. 跌破成本价5%设止损

### 风险控制

- 单只股票仓位不超过总资金20%
- 总体亏损达10%强制减仓
- 追高买入的股票必须设止损


### 计算VWAP（主力成本）

```python
# 查K线算VWAP
ret, kline = quote_ctx.request_history_kline(code, max_count=20)
kline['typical_price'] = (kline['high'] + kline['low'] + kline['close']) / 3
kline['amount'] = kline['typical_price'] * kline['volume']
vwap = kline['amount'].sum() / kline['volume'].sum()
```

### 查询持仓并分析

```python
# 查询真实持仓
ret, positions = trd_ctx.position_list_query()

for _, pos in positions[positions['qty'] > 0].iterrows():
    code = pos['code']
    cost = pos['cost_price']
    current = pos['nominal_price']

    # 获取VWAP
    ret, kline = quote_ctx.request_history_kline(code, max_count=20)
    vwap_20 = calculate_vwap(kline)

    # 比较成本
    if cost < vwap_20:
        print(f"{code}: 优势持仓，成本低于主力")
    else:
        print(f"{code}: 注意风险，成本高于主力")
```

## 八、总结

主力成本是判断买卖点的重要参考，但要结合消息面、技术面综合判断：

1. **成本低于主力** → 持仓更有优势，可以更有耐心
2. **成本高于主力** → 需要更谨慎的风险管理
3. **主力被套时** → 往往有护盘动力，可能是买入机会
4. **主力大幅盈利** → 注意出货风险，考虑止盈

**记住：富途的"主力"只是大单，不等于机构。真正的机构动向要看CCASS数据。**
