## 系统概述

### 2. 炒作指数计算 (0-100分)
```
炒作指数 = 平均涨幅×40% + 上涨家数占比×30% + 强势股占比×30%
```

评分标准:
- **80-100分**: 🔴 高潮期 - 疯狂炒作，注意风险
- **60-80分**: 🟠 加速期 - 快速上涨，跟随龙头
- **40-60分**: 🟢 启动期 - 早期机会，可以布局
- **0-40分**: ⚪ 冷淡期 - 关注度低

### 3. 板块生命周期识别
自动跟踪板块历史数据，判断当前处于:
- **启动期**: 炒作刚开始，早期介入机会
- **加速期**: 快速上涨，龙头引领
- **高潮期**: 疯狂阶段，随时见顶
- **高位震荡**: 炒作指数下降，准备退出

### 4. 龙头股&补涨机会
- **龙头识别**: 自动找出板块内涨幅最大的股票
- **补涨机会**: 当龙头涨幅 - 掉队股涨幅 > 5%时提示

### 5. 自动推送 (可选)
当检测到以下信号时，推送到Telegram:
- 🔥 炒作指数≥70 + 加速期/高潮期
- 💚 炒作指数≥50 + 启动期 (早期机会)

## Telegram推送配置 (可选)

### 步骤1: 创建Telegram Bot
1. 找 @BotFather 对话
2. 发送 `/newbot`
3. 按提示设置名称，获得Token

### 步骤2: 获取Chat ID
1. 打开你的Bot，发送 `/start`
2. 访问: `https://api.telegram.org/bot<你的Token>/getUpdates`
3. 找到 `"chat":{"id":123456789}` 中的数字

### 步骤3: 填入配置
编辑 `sector_radar.py` 第77-78行:
```python
TELEGRAM_BOT_TOKEN = "你的Bot Token"
TELEGRAM_CHAT_ID = "你的Chat ID"
```

### 步骤4: 重启服务
```bash
launchctl unload ~/Library/LaunchAgents/com.mantou.sector-radar.plist
launchctl load ~/Library/LaunchAgents/com.mantou.sector-radar.plist
```

## 技术细节

### 依赖

- futu-api (富途OpenD)
- requests (Telegram推送)

### 环境变量
```

```

### 重要说明
1. 需要富途OpenD运行在 127.0.0.1:11111
2. 自选股标记功能需要牛牛号登录(当前已禁用)
3. 仅读取行情数据，不涉及交易操作
