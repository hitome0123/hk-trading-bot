**改进资讯抓取**：
1. 不依赖外部新闻，**用量价数据推断**
2. 逻辑：
   ```
   涨幅 > 10% + 换手率 > 15% = 疑似重大利好
   成交额暴增 3倍+ = 资金抢筹
   连续3天上涨 = 趋势形成
   ```

3. 实现：
   ```python

       """从量价推断隐含信息"""

           return f"{stock['name']}大涨{stock['change_pct']:.1f}%且高换手，疑似重大利好未公开"
   ```

def search_news_with_gemini(stock_name):
    """让Gemini搜索最新新闻"""
    prompt = f"""
    搜索港股{stock_name}今天的最新新闻，
    尤其是：订单、业绩、政策、并购相关消息。
    返回JSON格式的摘要。
    """
    # Gemini 2.5可以联网搜索
    result = gemini_client.search_and_generate(prompt)

        # 1. 富途公告（最权威）
