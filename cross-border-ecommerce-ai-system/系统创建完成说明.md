#### 主Workflow
**名称**: 跨境电商全栈智能运营系统
**ID**: `94680151-0c7a-4b44-8e9a-c3be500e0174`
**状态**: 已激活 (Active)
**节点数**: 27个核心节点
**功能**: 完整的跨境电商自动化流程

**Workflow层次**:
```
触发层 (4节点)
  ├─ 定时触发器 (每10分钟)
  ├─ Webhook新订单
  ├─ 供应商邮件监控
  └─ 手动触发

数据聚合层 (5节点)
  ├─ 获取Amazon订单
  ├─ 获取eBay订单
  ├─ 获取Shopify订单
  ├─ 查询待处理订单 (PostgreSQL)
  └─ 检查Redis缓存

处理层 (5节点)
  ├─ 合并所有订单
  ├─ 数据清洗标准化
  ├─ 筛选高价值订单
  ├─ 分支按平台
  └─ 循环处理订单

AI决策层 (3节点)
  ├─ 调用智能客服Agent
  ├─ 调用动态定价Agent
  └─ 调用库存管理Agent

执行层 (5节点)
  ├─ 聚合AI决策
  ├─ 更新数据库
  ├─ 更新平台状态
  ├─ Telegram告警
  └─ 审计日志

监控层 (2节点)
  ├─ 审计日志记录
  └─ 性能指标统计
```
