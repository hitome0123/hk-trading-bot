## 🔧 配置步骤（待完成后执行）

### 第1步：申请X API
1. 访问 https://developer.twitter.com/
2. 创建开发者账号
3. 创建Project和App
4. 获取Bearer Token

### 第2步：配置环境变量
```bash
# 编辑 ~/.zshrc
nano ~/.zshrc

# 添加
export X_BEARER_TOKEN="your-bearer-token-here"

# 重新加载
source ~/.zshrc
```

### 第3步：测试X API
