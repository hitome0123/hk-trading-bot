# 港股低频稳健波段交易系统

**核心目标：5万本金，每天稳定赚300元（0.6%日收益），逐步进阶到500-1000元/天**

## 🎯 系统概览

### 你的基本参数
- **市场**：港股（T+0优势）
- **本金**：5万港元
- **策略**：低频稳健（每天1-2次交易）
- **初期目标**：日赚300元（0.6%）
- **进阶目标**：500元/天 → 1000元/天
- **单次仓位**：70%（3.5万）
- **单次盈利目标**：5-7%
- **最大单日亏损**：2000元（红线）
- **持仓策略**：可以长拿，但必须有止损纪律

### 两阶段进阶路径

**阶段A：基础稳健（前3个月）**
- 目标：日赚300元，胜率60%+
- 方法：Larry Williams三步骤系统
- 频率：每天1-2次交易
- 重点：纪律、风控、稳定性

**阶段B：进阶激进（3个月后）**
- 目标：日赚500-1000元
- 方法：港股T+0多次交易
- 频率：每天3-5次交易
- 重点：节奏、复利、快进快出

## 📋 阶段A：基础稳健系统（开始这里！）

### Larry Williams 三步骤交易系统

#### **步骤1：寻找市场（选股）**

每天开盘前30分钟完成选股，只选3-5只候选股票。

**选股标准（必须同时满足）：**

1. **技术面筛选**
   - ✅ 股价在上升趋势中（5日均线 > 10日均线 > 20日均线）
   - ✅ 成交量温和放大（不要暴量，暴量可能是出货）
   - ✅ MACD金叉或即将金叉
   - ✅ 股价在重要支撑位之上

2. **基本面辅助（避雷）**
   - ❌ 排除：财务造假、退市风险、ST股
   - ❌ 排除：近期有重大利空消息
   - ❌ 排除：成交量极小的仙股（容易被操控）

3. **港股特色筛选**
   - ✅ 优选：AH股（有A股联动，流动性好）
   - ✅ 优选：恒生指数成分股（波动相对可控）
   - ✅ 优选：你熟悉的行业（如手术机器人、新能源等）

**你的优势领域**：
- 手术机器人（微创机器人等）
- 短视频平台（快手等）
- 其他你看好的未来趋势行业

**实战工具**（涨乐财富通）：
- 自选股：每天只关注10-15只
- 技术指标：K线、均线、MACD、成交量
- 资金流向：查看主力资金是流入还是流出

---

#### **步骤2：伺机进入（买点）**

**核心原则：开盘价之下不买入，开盘价之上不卖出**（Larry Williams）

**最佳买入时机（4选1）：**

**买点1：跌破均线后的反弹（顺向T+0）**
- 股价短期跌破5日/10日均线
- 成交量萎缩（说明抛压减轻）
- 出现反弹信号（小阳线、下影线）
- **进场**：反弹突破均线时买入
- **止损**：跌破近期低点

**买点2：突破重要阻力位**
- 股价长期在某个价位横盘
- 突破时成交量明显放大
- **进场**：突破确认后买入
- **止损**：跌破突破位

**买点3：早盘急跌后的反转**（港股T+0优势）
- 开盘后30分钟内急跌3-5%
- 跌到重要支撑位（如前低、均线）
- 出现反转信号（止跌企稳、成交量萎缩）
- **进场**：反转确认后买入
- **止损**：跌破支撑位

**买点4：大阳线突破**（价格爆炸点）
- 前一天收大阳线（涨幅3%+）
- 今天开盘继续走强
- **进场**：确认趋势延续时买入
- **止损**：跌破大阳线低点

**仓位管理：**
- 单次买入：3.5万港元（70%仓位）
- 剩余1.5万：应急资金（补仓或新机会）

---

#### **步骤3：伺机退出（卖点）**

**三种退出方式：**

**退出1：达到目标价（止盈）**
- **初期目标**：5-7%（对应300-500元利润）
- **进阶目标**：7-10%（对应500-700元利润）
- **卖出时机**：
  - 股价达到目标价附近
  - 出现滞涨信号（量价背离）
  - 上升趋势减弱

**退出2：止损点（风控红线）**
- **初级止损**：-7%（亏损2450元）
- **专业止损**：-8%（Larry Williams建议，亏损2800元）
- **执行纪律**：
  - ⚠️ 触及止损线，立即卖出，不抱幻想
  - ⚠️ 不要"再等等看"，小亏扛成大亏是最大敌人

**退出3：跟踪止损（拿住利润）**

**解决你的"卖飞"问题：**

你之前的痛点：赚7%就卖，后来又涨10%+

**新策略：动态止盈**
1. 股价涨到5%：卖出50%仓位（锁定利润）
2. 剩余50%设置跟踪止损：
   - 从最高点回落3%就卖出
   - 这样既保护利润，又能拿住大肉

**示例**：
- 买入价：10港元，买入3.5万（3500股）
- 涨到10.5元（+5%）：卖出1750股，锁定利润875元
- 继续涨到11元：继续持有
- 从11元回落到10.67元（-3%）：卖出剩余1750股
- 总利润：875 + (10.67-10)×1750 = 875 + 1172.5 = 2047.5元

### 风险控制系统（最重要！）

#### **1. 单日最大亏损：2000元（红线）**

**执行规则：**
- 单笔亏损超过1000元 → 暂停30分钟，冷静
- 累计亏损达到2000元 → 立即停手，今天不再交易
- 连续3天亏损 → 暂停交易3天，复盘反思

**为什么2000元？**
- 占本金的4%，在可承受范围内
- 连续亏5天（极端情况）也只亏1万（20%）
- 给你足够的"试错"空间

#### **2. 被套后的处理（你的核心问题）**

**判断标准：长拿 vs 止损**

**可以长拿的情况（3个条件都满足）：**
1. ✅ 基本面没有恶化（公司正常经营，没有重大利空）
2. ✅ 上升趋势未破坏（5日均线未跌破20日均线）
3. ✅ 亏损在可控范围（≤8%）

**必须止损的情况（任一条满足）：**
1. ❌ 基本面恶化（财务造假、行业政策利空、公司出事）
2. ❌ 趋势彻底破坏（跌破所有均线，MACD死叉）
3. ❌ 亏损超过8%（触及止损红线）

**你的微创机器人案例分析：**
- 买入理由：看好手术机器人未来
- 卖出时机：赚7%就卖了
- 后续：又涨了10%+

**新策略应对：**
1. 如果趋势完好（均线多头排列）→ 使用跟踪止损，不要7%就跑
2. 如果遇到回调（-3%~5%）→ 判断是技术回调还是趋势反转
3. 如果基本面未变 + 技术面支撑 → 继续持有，甚至加仓

### 每日操作流程（SOP）

**09:00-09:30 开盘前准备**
1. 查看全球市场（A股、美股昨夜表现）
2. 查看港股大盘走势（恒生指数）
3. 筛选3-5只候选股票
4. 制定今日交易计划（买什么、买多少、止损位）

**09:30-12:00 上午交易时段**
1. 观察候选股票走势
2. 等待买点出现（不要追高）
3. 买入后设置止损价（心里有数）
4. 如果早盘急跌，可考虑T+0操作

**12:00-13:00 午休复盘**
1. 如果已建仓，检查持仓状态
2. 如果盈利，决定是否部分止盈
3. 如果亏损，判断是否需要止损

**13:00-16:00 下午交易时段**
1. 继续观察持仓
2. 达到目标价，果断止盈
3. 触及止损线，果断止损
4. 尾盘前30分钟，决定是否隔夜

**16:00-17:00 盘后复盘**
1. 记录今日交易（买入价、卖出价、盈亏）
2. 分析成功/失败原因
3. 更新自选股列表
4. 准备明日交易计划

### 心理与纪律管理（85%的交易是心理！）

**Oliver Velez核心理念：知识+纪律=成功**

#### **常见心理陷阱与应对**

**陷阱1：卖飞后悔（你的痛点）**
- **症状**：赚了7%卖出，眼看它涨到15%
- **心理**：后悔、懊恼、下次想拿更久
- **应对**：
  - 记住：没人能卖在最高点
  - 使用跟踪止损，让市场告诉你何时卖
  - 盈利就是成功，不要贪婪

**陷阱2：亏损不止损（普遍问题）**
- **症状**：亏了5%想"再等等"，结果变成-15%
- **心理**：侥幸、不甘心
- **应对**：
  - 小亏是成本，大亏是灾难
  - 止损后可以再买回来（港股T+0优势）
  - 保护本金是第一优先级

**陷阱3：追涨杀跌**
- **症状**：看到股票大涨就追，大跌就割
- **心理**：恐慌、FOMO
- **应对**：
  - 只在计划内的买点进场
  - 涨了就看，不追
  - 跌了看趋势，不恐慌

**陷阱4：报复性交易**
- **症状**：亏了想快速回本，频繁操作
- **心理**：焦虑、急躁
- **应对**：
  - 触发单日亏损2000元，立即停手
  - 连续亏损3天，暂停3天
  - 复盘找原因，不着急

### 进阶策略：港股T+0多次交易

**目标提升：300元/天 → 500-1000元/天**

**方法转变：**
- 从每天1-2次 → 每天3-5次
- 从单次5-7% → 单次2-3%，多次累积
- 从70%仓位 → 50%仓位轮动

#### **T+0日内多策略**

**策略1：高开反弹做空**
- 早盘高开3%+，观察成交量
- 如果量未放大，是假突破
- 高点卖出（做空或卖出持仓）
- 回落后再买回

**策略2：日内反转**
- 上午急跌3-5%
- 中午止跌企稳
- 下午反弹时买入
- 尾盘卖出

**策略3：突破交易**
- 盘中突破关键阻力位
- 成交量配合放大
- 买入跟进
- 获利2-3%即走

**策略4：尾盘偷袭**
- 14:30后观察强势股
- 尾盘30分钟急拉
- 快速买入
- 次日高开卖出

**进阶仓位管理：**
- 单次交易：2.5万（50%）
- 同时持仓不超过2只
- 留出空间应对波动

### 场景1：理想交易（标准案例）

**情况**：
- 发现快手（01024）技术面良好
- 早盘跌破5日均线至68元后开始反弹

**操作**：

2. 买入3.5万 ÷ 68.5 = 511股
3. 设置止损：66.5元（-2.9%）
4. 目标价：72.3元（+5.5%）

**结果A（顺利）**：
- 14:30，股价涨至72.5元
- 卖出511股 × (72.5-68.5) = 2044元
- ✅ 超额完成今日目标

**结果B（止损）**：
- 10:30，股价跌破66.5元
- 果断止损，亏损 511股 × 2元 = 1022元
- 继续寻找下一个机会

### 场景8：市场极端波动

**情况**：
- 港股大盘暴跌5%
- 个股普遍跌停
- 你的持仓跌了3%

**操作**：
1. **不抄底**：暴跌时不要冲动买入
2. **评估持仓**：
   - 如果基本面OK，趋势未坏 → 持有
   - 如果技术面破位 → 果断止损
3. **降低仓位**：暂时空仓或轻仓
4. **等待企稳**：等大盘企稳后再进场

**原则**：
- 不逆势而为（Larry Williams）
- 保护本金第一
- 机会永远有，本金没了就没了

### 场景10：连续亏损后的焦虑

**情况**：
- 连续3天亏损，累计-3000元
- 产生"报复性交易"冲动
- 想快速回本

**心理陷阱**：
- "我必须今天赚回来"
- "加大仓位，赌一把"

**应对**：
- ⚠️ **立即停止**交易3天
- 复盘这3天的交易
- 找出亏损原因：
  - 是选股问题？
  - 是买点问题？

### 场景11：被套深度套牢

**情况**：
- 持有微创机器人，买在高位60元
- 现价51元，亏损15%（-4500元）

**判断流程：**

**步骤1：基本面判断**
- 手术机器人行业前景如何？
- 公司有无重大利空？
- 如果基本面OK → 继续

**步骤2：技术面判断**
- 是否跌破所有均线？
- MACD是否死叉？
- 成交量是否持续萎缩？
- 如果技术面破位 → 止损

**步骤3：仓位影响**
- 4500元亏损占本金9%
- 已经超过8%止损线
- **应该止损**，即使看好基本面

**决策**：
- **止损离场**，保护剩余本金
- 等股价企稳后再考虑重新买入
- 不要死扛，小亏变大亏是大忌

**如果你坚持长拿：**
- 必须降低仓位（比如只留30%）
- 设置更严格的止损（比如-20%彻底走）
- 定期复盘，每周评估一次
