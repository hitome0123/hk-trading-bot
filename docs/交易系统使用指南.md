## 🎓 技能核心内容速查

### Larry Williams 三步骤
1. **寻找市场**（选股）→ 用bot扫描
2. **伺机进入**（买点）→ 用技能判断
3. **伺机退出**（卖点）→ 用技能判断

### 风险控制红线
- 单日最大亏损：2000元（4%）
- 单笔止损：7-8%
- 仓位控制：70%（3.5万）

### 买入时机（4选1）
1. 跌破均线后反弹
2. 突破重要阻力位
3. 早盘急跌后反转
4. 大阳线突破

### 卖出时机（3选1）
1. 达到目标价（5-7%）
2. 触及止损（-7-8%）
3. 跟踪止损（从最高点回落3%）

### 被套处理判断
**可以长拿：**
- 基本面OK ✅
- 趋势未破 ✅
- 亏损≤8% ✅

**必须止损：**
- 基本面恶化 ❌
- 趋势破坏 ❌
- 亏损>8% ❌

## ⚙️ 配置建议

### Bot配置
编辑 `.t_recommendation_config.json`:
```json
{
  "capital": 50000,           // 本金5万
  "max_positions": 2,         // 最多持仓2只
  "min_profit_pct": 5,        // 最小盈利5%
  "max_loss_pct": 8,          // 最大亏损8%
  "daily_trade_limit": 5      // 每日交易限制
}
```

### 技能参数
- 单次仓位：70%（3.5万）
- 盈利目标：5-7%
- 止损红线：7-8%
- 单日亏损上限：2000元
