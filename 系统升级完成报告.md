## 🚀 新功能详解

### 方案A: 量价推断资讯（已集成）

**原理**：从价格和成交量推断隐含信息

**判断逻辑**：
```python
涨幅>15% + 换手率>20% = 重大利好催化（订单/并购/业绩）
涨幅>10% + 换手率>15% = 较强利好，资金积极参与
涨幅>5%  + 换手率>10% = 温和上涨，关注度提升
成交额>10亿          = 资金大幅流入
```

**优点**：
- ✅ 不依赖外部API
- ✅ 基于真实行情数据
- ✅ 立即可用

**示例输出**：
```
📈 优必选大涨12.3%且高换手15.2%，资金积极参与，可能有未公开利好
💰 优必选成交额5.0亿，资金大幅流入
```
